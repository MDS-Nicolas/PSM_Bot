Host Computer

Software Components

The Host Computer is backed-up using Acronis True Image Echo Server for Linux software. For details, see Data Backup and Recovery.

Prior to installing any additional software, consult MDS Aero Support Corporation.

The following table identifies the software components that are installed on the Host Computer:

Component Description
Red Hat Linux Enterprise 5 Linux operating system.
Red Hat MRG Linux Real-Time Kernal Software used for applications with high determinism requirements.
ORACLE 11g Relational Data Base Management System (RDBMS) Database software used for the storage of test results and calibrations.
proDAS Real-Time data acquisition software Real-time data acquisition software that monitors and controls the data acquisition process.
Acronis Backup & Restore Linux Server Software to backup data and recover backed-up data.

For OEM maintenance information, see Vendor Documentation.
