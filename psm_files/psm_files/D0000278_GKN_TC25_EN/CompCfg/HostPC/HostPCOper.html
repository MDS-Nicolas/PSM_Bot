Host Computer

Operation

The Host Computer runs a Real-Time multi-tasking Linux operating system which allows task priorities to be set in parallel, and ensures that important tasks are performed in a deterministic time frame. The Host Computer is installed in Equipment Rack CM1 in the Computer Room.

Basic computer operational instructions and application information are contained in the How To section.
