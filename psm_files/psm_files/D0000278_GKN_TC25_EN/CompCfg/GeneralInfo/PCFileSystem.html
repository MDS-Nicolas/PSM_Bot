Windows-based Computers

File System Organisation

The proDAS software is centrally located in a root proDAS directory. This directory can be stored on any drive letter.

A typical proDAS directory structure on drive C: is shown below:


The table below describes some of the typical folders found in the proDAS directory.

Folder name Description
Bin The proDAS software, read-only resource files and language files reside in the bin folder or in application specific bin folders (for example, bin-SensorCalibration).
Data The Data folder contains application specific data contained in sub-folders such as:
  • Configuration - contains the files that make up a master configuration
  • Mgt GUI - contains files that the Mgt GUI uses to run proDAS (such as scripts to start and stop proDAS scanning) and test procedures
  • Trace (all trace files) - trace files contain information which can be used for troubleshooting
  • ScriptEditor - contains test procedure libraries
  • Sensor Calibration - contains sensor calibration data
  • UserSecurity - contains user security database files
Ini The Ini folder contains all of the initialization (INI) files. Initialization files contain configuration parameters and layout information for each proDAS application such as the Channel Editor, Subsystem Editor, or Script Editor.
Results The Results folder contains the test results.
Tools The Tools folder contains tools used to troubleshoot issues with proDAS.
