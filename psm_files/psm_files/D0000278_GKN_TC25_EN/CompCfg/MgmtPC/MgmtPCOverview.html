Management GUI Computer

Overview

The Management GUI Computer (also referred to as Management Graphical User Interface Computer) is a PC-based computer with a Windows operating system. The Management GUI Computer communicates and interacts with the proDAS real-time software, the proDAS database, and individual hardware subsystems. The Management GUI Computer provides a graphical user interface to proDAS. The Management GUI Computer is located in the console in the Control Room.

Management GUI Control Panel functions are point-and-click activated. Each function on the Management GUI Control Panel is security controlled and sensitive to the login level of the user.

The Management GUI Computer performs both real-time and non real-time functions. Primary real-time tasks of the Management GUI Computer include:

  • receiving and displaying data
  • logging (storing) data to the database
  • sending requests to the Real-Time Engine (RTE)

Non-real-time tasks include:

  • configuring proDAS
  • running automated test scripts
  • calibration
  • recording and playing back data
  • generating post test reports
