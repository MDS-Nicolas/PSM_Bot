Data Backup & Recovery for Windows Based Computers

Overview

Proper data backup is key to preventing loss of information and reducing facility down-time.

All data on the Windows based computers is protected with Acronis Backup and Recovery Advanced Workstation with Universal Restore software which creates backup images of the local hard drive. The backup images are stored on a NAS (Network Attached Storage).

MDS recommends reviewing the backup log files weekly to verify successful completion of the backup operation. Failing to do so could result in permanent loss of data.

Acronis Backup and Recovery Advanced Workstation with Universal Restore software is also used to restore data.

The Acronis software is used for the following tasks:

  • Data Backup
  • Data Recovery
  • Disaster Recovery
  • Reviewing Scheduled Tasks
  • Reviewing Logs

Window Data Backup

Data backup tasks, as configured by MDS, are designed for maximum protection against data loss. MDS highly recommends that the following tasks not be changed:

  • Full backup on Saturday
  • Incremental back-ups Sunday through Monday inclusively

Typically full backups are performed once a week and are part of an overall back-up plan.

Each Saturday, the full backup replaces the back-up image created the previous Saturday. The incremental backup saves time and disk space by only backing up changed files. The incremental backups scheduled Sunday to Friday will only backup the data that has changed since the last full backup or last incremental backup. On Monday this will be the differences between the disk on Monday and Sunday. Tuesday's image is the difference between the disk on Tuesday and Monday. Incremental back-ups are scheduled through the week until Friday.

For the backup schedule, see Facility Computers Backup Schedule.

Rescue Bootable Media CD

As part of the backup strategy, it is recommended that an Acronis Bootable Rescue Media be created for the Windows based computers before the initial engine testing.

It is highly recommended that an Acronis Bootable Rescue Media be created using the Acronis Backup and Recovery Advanced Workstation with Universal Restore software before testing begins for the first time. The Acronis Bootable Rescue Media is an integral part of the recovery process. Failure to create the Acronis Bootable Rescue Media could result in permanent loss of data.

For further information on creating an Acronis Bootable Rescue Media, see the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.
