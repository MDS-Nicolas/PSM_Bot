Real-Time Display Computer

Overview

The Real-Time Display (RTD) Computer displays real-time data, such as engine test parameters. The RTD Computer is a PC based computer with a Windows Operating System.

There are typically multiple RTD Computers, set up in various locations such as:

  • on the Operator Console in the Control Room
  • on the OEM Console in the Control Room
  • in the main table in the Monitoring Room

The RTD computer is connected to one or more monitors mounted on their respective consoles.

The RTD Computer screens display engine test parameters. The control menus and buttons on the screens interface with proDAS real-time software to:

  • display and change operator pages
  • create and modify operator pages (using the RTD Editor)
  • print Real-Time screens
  • use Spotlight Stickies to view and set channel values
  • monitor the engine test
  • view the Event Handler window
  • view and monitor alarms
