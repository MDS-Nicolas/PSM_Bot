Real-Time Display Computer

Software Components

The following table identifies the software components that are installed on the Real-Time Display Computer:

Component Description
Windows 10 Enterprise Operating system.
Microsoft Office Basic 2010 Microsoft Office 2010 is a suite of applications.
ORACLE 11g Client Software to extract data from the Test Result and Sensor Calibration Database on the Host Computer.
Acronis Backup and Recovery Data backup and recovery software.
DataViews 2.9 Runtime A data visualization application that contains a series of customizable graphic development tools. The software is used to create and display the Real-Time Display (RTD) pages.
XML Notepad 2007 Software for browsing and editing XML documents on the Management GUI Computer.
Cygwin Cygwin is a large collection of GNU and Open Source tools which provide functionality similar to a Linux distribution on Windows. A DLL (cygwin1.dll) provides substantial POSIX API functionality. The software is used to access the Real-Time Display Computer (RTD) with XTerminal.
PDF Complete Software for creating, editing and viewing PDF documents.
Trend Micro Antivirus software.
Softing OPC Connect Softing OPC Easy Connect software is used to translate OPC-DA communication protocol to OPC-XML-DA. It is required for the OPC Subsystem to communicate with devices using only the OPC-DA communication protocol.
The Softing OPC Connect software is only installed on the RTD 4 Computer.
WIBU Key A dongle that contains an MDS software key which allows MDS software applications to operate on the computer.
proDAS Application Software for the proDAS applications on the Real-Time Display computer.

For OEM maintenance information, see Vendor Documentation.
