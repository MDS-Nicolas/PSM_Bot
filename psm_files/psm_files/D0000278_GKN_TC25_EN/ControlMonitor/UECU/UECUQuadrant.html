Throttle

Throttle Quadrant

The throttle quadrant is the user interface to the engine fuel control system. The throttle quadrant is a throttle lever and signal conditioning system that automatically adapts to the full motion of the fuel control system. The throttle quadrant is equipped with one or two throttle levers and a touchscreen display computer. The throttle quadrant is located on the console in the Test Cell control room.

Throttle Lever

The throttle lever is a mechanical linear slider. The throttle lever permits the operator to control the position of the Throttle actuation unit located on or near the engine. The position of the throttle lever has a direct relationship to the position of the actuator/resolver/RVDT. Moving the throttle lever toward the console moves the actuator/resolver/RVDT toward the maximum throttle position. Moving the throttle lever toward the operator moves the actuator/resolver/RVDT to the minimum throttle position.

For dual lever systems, a second slider assembly is added to the left of the quadrant. The second lever functions the same as the first lever.

Display Computer

The display computer is a 10.4 inch panel-mounted touchscreen computer integrated into the throttle quadrant. The display computer runs the Throttle control display software using the Microsoft Windows 7 platform.

The display computer has the following features:

  • touchscreen buttons and controls
  • LCD screen to show Throttle position and other pertinent information

The Throttle control display software is a Graphic User Interface (GUI) which permits the operator to control the Throttle System.

The display software allows for menu driven and/or automatic configuration of test engines and connected devices.

The display computer uses Acronis Backup and Recovery with Universal Restore for Advanced Workstation software. For details, see How to Backup and Recover Data.
