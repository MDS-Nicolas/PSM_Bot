Throttle

Overview

The Universal Engine Command Unit (UECU-500 Mark III) (also know as the Throttle System) is a modular throttle actuation system intended for electrical and/or mechanical control of an engine fuel system. The UECU-500 Mark III can accommodate a resolver, rotary variable displacement transducer (RVDT), mechanical actuator, or a combination of the three.

The UECU-500 Mark III permits an operator to precisely control the position of a fuel control input shaft, or any similar mechanical device mounted in a Test Cell or stationary engine installation.

The UECU-500 Mark III Throttle features include:

  • easy installation into existing facilities
  • custom configuration (can be configured as per customer requirements)
  • single or dual lever configuration
  • adjustable Throttle lever stops
  • LCD touch screen
  • menu driven
  • automatic Throttle control

The dual lever variant can be configured to control a propeller or dynamometer.

The UECU-500 Mark III Throttle consists of the following main components and their connecting cables:

  • Required Components
    • the throttle quadrant (either single or dual)
    • the controller chassis
  • Optional Components
    • actuator control
      • the manual override panel
      • the actuator chassis
      • the mechanical actuator
      • the pneumatic control junction box
    • resolver control
      • the resolver chassis
    • RVDT control
      • the RVDT chassis
