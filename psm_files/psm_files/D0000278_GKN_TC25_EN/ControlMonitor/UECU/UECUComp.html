Throttle System Components

The Throttle System contains the following major components:

Component Description
Throttle Quadrant The throttle quadrant is the user interface to the engine fuel control system, typically located on the console in the Test Cell control room
Throttle Controller The throttle controller contains a PLC with discrete and analogue input/output modules, which controls the test engine based on inputs from the throttle quadrant.
Resolver Chassis (if required) Resolvers are typically included with systems that have engines equipped with electronic fuel control.
RVDT Chassis (if required) RVDT's are typically included with systems that have engines equipped with electronic fuel control.
Actuator (if required) An actuator is typically included with systems that have engines that require electromechanical fuel control.
Remote Chassis (if required) The remote chassis is an Ethernet IP remote I/O adapter used to communicate with the throttle quadrant. It is used in Throttle System installations where there is a long distance between the throttle quadrant in the control room and the engine where the actuator/resolver/RVDT control is performed.

The Throttle Controller is located in an equipment rack closer to the engine than the throttle quadrant.
Touchscreen Display Computer Software Software configuration of the touchscreen display computer.
Touchscreen Display Computer Hardware Hardware configuration of the touchscreen display computer.
