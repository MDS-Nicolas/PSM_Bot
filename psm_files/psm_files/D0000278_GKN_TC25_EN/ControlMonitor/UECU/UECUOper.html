Throttle

Operation

The Throttle System allows an operator to accurately control the fuel supply to a test engine. The Throttle System can be run in manual mode or auto throttle mode. In manual mode the operator controls the fuel supply to the test engine manually using the throttle lever(s). In auto throttle mode, the fuel supply to the engine is controlled automatically based on a selected feedback signal from the engine. For more details, see How To – Run an Engine in Auto Throttle Mode.

Configuration

The Throttle System uses configuration files to determine the configuration of the system components. The configuration files are .xml files that are stored in the display computer in the throttle quadrant. The applicable configuration is loaded from the configuration files when an engine is selected on the display computer.

Fuel Valve Control

The Throttle System can accommodate a resolver, rotary variable displacement transducer (RVDT), mechanical actuator, or a combination of the three to control the fuel valve on an engine.

Actuator

An actuator uses a Proportional, Integral, Derivative (PID) control loop to control the fuel valve. The PID control loop continuously calculates the difference between a desired setpoint (e.g. commanded fuel valve position) and the measured variable (e.g. measured fuel valve position). The throttle controller attempts to minimise the error by adjusting the control variable for the position of the fuel valve.

The PID control loops for the actuators take the lever command as the setpoint for the PID control loop. The lever command is sent in degrees to the actuator's PID routine in the throttle controller. This PID setpoint is limited and scaled according to the PID minimum and maximum settings in the master configuration file.

Resolver/RVDT

The Resolver and RVDT actuator types do not need a PID control loop to control the fuel valve. The lever command is sent to the RVDT/Resolver scaling routine in the throttle controller. The throttle controller scales the output as required and converts the output to 14 bit binary signals. The signals are sent to the TTL output card in the throttle controller.

The resolver and RVDT both have two outputs to the engine's electronic engine control (EEC). The outputs can both be scaled using the same scaling factors or each output can have its own scaling factors. This is configured directly in the master configuration file.

Using the Throttle System

The procedure below, provides a quick overview on using the throttle system.

To prevent injury and damage to equipment, only trained operators who are familiar with all applicable procedures at the engine testing facility and the procedures in this manual should operate the throttle system.

  1. Start the Throttle System and wait for it to boot up. See How To – Start the UECU-500 Mark III Throttle.
  2. Using the display computer in the throttle quadrant, select an engine. See How To – Select an Engine.
  3. If applicable, rig all actuators defined for the engine. See How To – Rig an Actuator.
  4. Enter Run Mode and turn on the fuel supply to the Engine. See How To – Run an Engine in Run Mode (Manual).
  5. Start the engine using the particular engine testing facility procedures.

    6. It is not possible to start the engine using the throttle quadrant.

  6. Use the throttle lever(s) on the throttle quadrant to control the engine under test.
  7. Use the Trip buttons to move to any preset trip points.
