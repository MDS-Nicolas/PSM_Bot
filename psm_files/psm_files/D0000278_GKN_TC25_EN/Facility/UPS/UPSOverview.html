Uninterruptible Power Supply (UPS)

Overview

The Uninterruptible Power Supply (UPS) provides emergency battery back-up power for data systems and for test and control equipment. The UPS also ensures that the back-up power is free of surges, sags and transients, allowing test data to be saved and the engine to be shutdown safely if a facility electrical power failure occurs.

Typical systems that a UPS provides back-up power for are:

  • proDAS
  • Facility Control System (FCS)
  • Engine Control System (ECS)
  • Throttle System
  • Dynamic Data System
  • CCTV System
  • Intercom System
  • Safety Lockout and Emergency Stop System
