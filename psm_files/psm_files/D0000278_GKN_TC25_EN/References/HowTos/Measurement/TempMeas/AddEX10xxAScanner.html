How To - Add an EX10xxA Temperature Scanner

This How To describes the required steps to add an EX10xxA temperature scanner to the proDAS system.

When a new VEXA EX10xxA scanner is being added to the proDAS system, the IP address for the scanner needs to be entered in the Linux PC hosts file.

To Add a EX10xxA Temperature Scanner

  1. Login to the Linux PC as the root user and use a text editor (nedit, gedit or vi) to open the /etc/hosts file for editing.
  2. Scroll down in the file to where the Real-Time Network IP addresses are defined.
  3. Search for any previously defined EX10xxA scanners.

    Normally the host name for a VEXA EX10xxA scanner has the following naming convention: VEXA<serial_number>

    If the serial number for the EX10xxA scanner is 209451, then the host name would be entered as VEXA209451. A previously unused IP address must be allocated to the scanner.

  4. Review the currently allocated IP addresses in the hosts file for all the devices on the Real-Time Network and select one that is not assigned to any device.
  5. Using an example of an unallocated IP address of 172.29.18.44, the following line would be added to the /etc/hosts file in the same location where the previously defined EX10xxA scanners are found:

    172.29.18.44    VEXA209451

  6. Save the /etc/hosts file and logout of the Linux PC.
  7. On the MgtGUI PC, launch an instance of the Subsystem Editor.
  8. Select the VEXA Subsystem. A list of the currently defined EX10xxA scanners is displayed on the right-hand side.
  9. Right-click on the right-hand side and select the Append option to add a new scanner.
  10. In the Host Name field, enter the same host name that was entered in the /etc/hosts file.
  11. Enter the remaining information based on the type of scanner that is being added – model number, unit scan rate and, default EU.
  12. Continue and add the Modules/Channels information for this new scanner.
  13. Save the changes in the Subsystem Editor. You can now add VEXA channels in the Channel Editor and assign them to this new EX10xxA scanner.
