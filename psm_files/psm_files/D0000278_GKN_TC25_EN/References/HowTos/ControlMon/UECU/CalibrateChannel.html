How To – Calibrate a Channel

This How To provides the steps required to calibrate a channel.

All analogue input channels must be calibrated to convert raw voltage signals into appropriate engineering units.

Analogue input channels should be re-calibrated once a year or after any system maintenance.

Remote Command and the Feedback 1 to Feedback 5 channels are only be available if configured in the Channel Calibration Data Configuration file (calibrationData.xml).

This How to covers the following topics:

  • Interlocks
  • Channels
  • To Calibrate a Channel
    • Lever Channel Calibration
    • Actuator Channel Calibration
    • Analogue Channel Calibration

Interlocks

To calibrate a channel, the user must press the Calibration button to enter the Calibration screen.

The Calibration button is disabled (greyed out) if any of the following interlock conditions are TRUE:

  • The Throttle system is in Auto Throttle Mode
  • The fuel is on
  • The calibration is disabled via a hardwired discrete input
  • Rigging is in progress
  • An engine is not selected
  • An engine selection is in progress

To Calibrate a Channel

There are three different channel types. Each channel type has a different calibration procedure based on the methods used to change the raw signal type.

  • Lever Channel Calibration
  • Actuator Channel Calibration
  • Analogue Channel Calibration

Lever Channel Calibration

To calibrate a lever channel:

  1. At the bottom of the display computer screen, press the Calibration button.
    The Calibration screen is displayed.

    2. If the calibration button is disabled, one or more interlocks are in place. For details, see Interlocks.


  2. In the Select Channel list box, select the Right Lever. (If is is a dual lever system, select the lever channel to calibrate, either Right Lever or Left Lever).
  3. Press the Start button.
  4. Manually move the applicable lever (right or left) to its minimum position (nearest the operator).


  5. Press the Accept button.
    The display computer displays the current point from the Throttle Controller.
  6. Manually move the applicable lever (right or left) to its maximum position (away from the operator).
  7. Press the Accept button.
    The display computer displays the current point from the Throttle Controller.
  8. Press the Save button to save the calibration.

    9. If the Cancel button is pressed, the data is cleared and not applied.

Actuator Channel Calibration

To calibrate an Actuator channel:

  1. At the bottom of the display computer screen, press the Calibration button.
    The Calibration screen is displayed.

    2. If the calibration button is disabled, one or more interlocks are in place. For details, see Interlocks.

  2. In the Select Channel list box, select the Right Actuator. (If it is a dual lever system, select the actuator channel to calibrate, either Right or Left Actuator.)
  3. Press the Start button.
    • The Throttle Controller automatically moves the Actuator to the low point at a slow controlled rate and holds until the signal stabilizes
    • The Throttle Controller automatically moves the Actuator to the high point at a slow controlled rate and holds until the signal stabilizes
    • The Throttle Controller calculates the new breakpoint table
    • The new low and high point values are displayed.


  4. Press the Save button to save the calibration.

    5. If the Cancel button is pressed, the data is cleared and not applied.

Analogue Channel Calibration

In an Analogue Channel Calibration, DC voltages represent an engineering unit. Engineering units can be values such as lever angle, speed or thrust. Before starting the procedure below, determine what voltages will represent each engineering unit. The table below outlines an example.

DC Voltage Engineering Unit (Speed)
0 VDC 0 RPM's
2.5 VDC 2000 RPM's
5 VDC 4000 RPM's
7.5 VDC 6000 RPM's
8 VDC 8000 RPM's

DC voltages should be injected into the J6 connector on the back of the Throttle Controller using a voltage simulator or by simulating the actual EU value. For pin details for each channel, see Customer Interface connections.

To calibrate an analogue channel you modify the Channel Calibration Data configuration file. Before modifying this configuration file you must make a copy of the file as a backup in case you have to revert to this known configuration.

Only channels between the <autoChannels></autoChannels> elements can be configured using the procedure below.

To calibrate an analogue channel:

  1. On the display computer, navigate to the Channel Calibration Configuration file. (For details, see How To – Modify the Configuration Files.)
  2. Scroll down to the bottom of the file.
  3. Enter the desired engineering units between the EU channel elements for the channel that is being calibrated. In this example, <fb1EU><Values>0 2000 4000 6000 8000</Values></fb1EU>.

    4. The engineering units could be values such as engine speed or thrust.

    The engineering units must go from the smallest value to the largest value (example: 0 2000 4000 6000 8000). The Throttle System will reject the following: 0 4000 2000 8000 6000.


  4. Save and close the file.
  5. Restart the display computer GUI software. For details, see Starting the UECU-500 Mark III Throttle.
  6. Select the desired engine. For details, see Selecting an Engine Using the Throttle Quadrant.
  7. At the bottom of the display computer screen, press the Calibration button.
    The Calibration screen is displayed.

    8. If the calibration button is disabled, one or more interlocks are in place. For details, see Interlocks.

  8. In the Select Channel list box, select an analogue channel to calibrate (Remote Command or Feedback#). In this example, Feedback 1.
  9. Press the Start button.
  10. Using a voltage source (or by simulating the EU value at the source), inject a DC voltage that represents the first Engineering Unit in the Target column on the Calibration Mode screen.
    The Throttle Quadrant displays the Raw voltage injected into the Throttle Controller.


  11. Press the Accept button.
    The Throttle Quadrant displays the next Target.


  12. Using a voltage source (or by simulating the EU value at the source), inject a DC voltage that represents the next Engineering Unit in the Target column on the Calibration Mode screen.
    The Throttle Quadrant displays the Raw voltage injected into the Throttle Controller.
  13. Press the Accept button.
    The Throttle Quadrant displays the next Target.
  14. Repeat steps 12 and 13 until complete.

    15. A maximum of 8 points can be injected. The number of points has to match number of engineering units you entered in step 2. In this example, 5 engineering units were entered in step 2, so it is a five point calibration.

  15. Press the Save button to save the calibration.

    16. If the Cancel button is pressed, the data is cleared and not applied.
