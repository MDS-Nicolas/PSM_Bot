How To – Backup and Recover Data for the UECU-Mark III Throttle System

This How To provides information on how to backup and recover data for the UECU-500 Mark III Throttle System.

Overview

This section is applicable to customers who purchased the UECU-500 Mark III Throttle System without an MDS proDAS installation. For customers with an MDS proDAS installation, refer to the Data Back-up and Recovery procedures in the Computer Config section of your PSM.

MDS recommends reviewing the backup log files weekly to verify successful completion of the backup operation. Failing to do so could result in permanent loss of data.

The UECU display computer is backed up using Acronis Backup and Recovery Advanced Workstation with Universal Restore software. Acronis Backup and Recovery Advanced Workstation with Universal Restore automatically creates backup images of the display computer's hard drive. The backup images are automatically stored on an external source such as a Network Attached Storage (NAS).

Performing a backup, restoration or a crash recovery of data is all accomplished with Acronis Backup and Recovery Advanced Workstation with Universal Restore software. Acronis allows the user to perform the following:

  • Starting Acronis Backup and Recovery Advanced Workstation with Universal Restore
  • Reviewing Scheduled Tasks
  • Reviewing Logs
  • Cancelling a Back-up
  • Data Back-up
  • Data Recovery
  • Disaster Recovery

As part of an effective backup strategy, it is recommended that an Acronis Bootable Rescue Media be created for the display computer before testing begins for the first time.

It is highly recommended that an Acronis Bootable Rescue Media be created using the Acronis Backup and Recovery Advanced Workstation with Universal Restore software before testing begins for the first time. The Acronis Bootable Rescue Media is an integral part of the recovery process. Failure to create an Acronis Bootable Rescue Media could result in permanent loss of data.

For further information on creating an Acronis Bootable Rescue Media , see the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.

Acronis Backup and Recovery Advanced Workstation with Universal Restore is very powerful software for data backup and recovery. It has many useful functions that are not covered by this manual. For more information, see the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.

Starting Acronis Backup and Recovery Advanced Workstation with Universal Restore

The following procedure contains the steps to start the Acronis Backup and Recovery Advanced Workstation application.

To start Acronis Backup and Recovery Advanced Workstation with Universal Restore:

  1. Double-click the Acronis icon (shortcut) located on the desk top of the applicable management computer.
    Acronis Backup and Recovery opens.
  2. From the Acronis Backup and Recovery Management Console, select Connect to a management server.


  3. In the Management Server Connection window, enter the applicable management computer name in the Machine field.
  4. Select OK.


  5. In the Access Credentials window, login as Administrator and enter a password for the Administrator’s account.
  6. Select OK.


Reviewing Scheduled Tasks

Using Acronis, it is possible to review the existing scheduled tasks.

To review scheduled tasks:

  1. Select Backup Plans and Tasks from the Navigation panel on the left side of the main Acronis window.
    The list of scheduled backup plans and tasks is displayed.


Use the panel buttons to create, run, edit, delete and view details of the backup plans and tasks. For further details, see the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.

Reviewing Logs

Every time Acronis attempts a backup, whether it is successful or not, a log of the action is created.

To review the logs:

  1. Select Log from the Navigation panel on the left side of the main Acronis window.
    The Log panel is displayed.
  2. From the Display menus, select the desired tasks and the period you want displayed.
    The list of Acronis tasks for the date(s) selected are presented.
  3. From the Log tab, select the desired log.
    The log Entry Details window is displayed.


  4. Select Close from the Log Entry Details window.

MDS recommends that the user review log files to confirm a successful backup.

Cancelling a Back-up

If testing is going to be performed while a backup is scheduled to take place, the backup can be manually cancelled. A backup can only be cancelled when logged in as an Administrator.

To cancel a backup process:

  1. Log into the applicable management computer. See Starting Acronis Backup and Recovery Advanced Workstation with Universal Restore.
  2. Select Backup Plans and Tasks from the Navigation panel on the left side of the main Acronis window.
    The Backup Plans and Tasks panel is displayed.


  3. Select a running backup task.

    4. To verify the task selected is the desired task, review the task details at the bottom of the screen to ensure the proper task has been selected.

  4. In the Back Up Plans and Tasks panel, select the Stop button.

For more information see, the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.

Manual Backups

Manual backups can be performed if desired. For details, see the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.

Data Recovery

In the unlikely event that a file is inadvertently deleted, use the following procedure to recover the data.

To recover data by mounting an image:

  1. Log into the applicable management computer as Administrator. See Starting Acronis Backup and Recovery Advanced Workstation with Universal Restore.
  2. Using Windows Explorer, browse to the location your backup images are stored.
  3. Select a backup image and right click on the selection.


  4. Select Mount from the context menu.
    The Access Credentials window is displayed.


  5. In the Access Credentials window, login as engineer and enter a password for the engineer's account and then select OK.
    Acronis Backup & Recovery opens and the Mount Image panel is displayed.


  6. In the Mount Image panel, select Source backup.
    The Backup selection window is displayed.


    7. MBR must be selected or the computer might not boot and Universal Restore.

  7. In the Backup selection window, select a backup.
  8. Select OK to close the Backup Selection window and return to the Mount image panel.


  9. In the Settings section of the mount image panel, select Change.
    The Volume Selection window is displayed.


  10. Select the checkbox of the driver partition (in this example, NTFS(C:)) and assign a letter to the new virtual drive.
  11. Select OK to close the Volume Selection window and to return to the Mount image panel.
  12. Select OK.
    The Manage mounted image panel is displayed.


  13. In the Manage mounted images panel, select Explore.
    The virtual drive containing the mounted files is displayed.


  14. Recover the desired files from the virtual drive (in this example, drive E).
  15. Unmount the virtual drive E by selecting the virtual drive letter and clicking on Unmount in the Manage mounted images panel.

To restore a file from an incremental backup, the referenced full backup is also required.

For more information see, the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.

Disaster Recovery

Acronis Backup and Recovery Advanced Workstation with Universal Restore software also provides the capability to perform bare metal crash recovery in the event that the computer operating system does not start-up normally. It is possible to start the restoration process without loading the operating system and then to proceed with restoring the damaged partitions.

In the unlikely event that the local hard drive or operating system fail, use the following procedure to recover the data.

To recover data:

  1. In the case of a hard drive failure, start by replacing the damaged hard drive with a hard drive that has the same or greater capacity than the damaged hard drive.
  2. Press F9 and verify that Boot from CD is the first item on the list. Boot the damaged computer using the Acronis Bootable Rescue Media.

    3. In the event that an Acronis Bootable Rescue Media for Windows is not available or has been misplaced, the Acronis Bootable Rescue Media can be created using another computer which is running Acronis.

  3. Insert the bootable media (CD or DVD) into the “damaged” computer and reboot the computer.

    4. A screen with three (3) options may be displayed. The options are: continue Acronis with a 32 bit system, continue Acronis with a 64 bit system, or continue booting the OS from the hard drive. Select either 32 or 64 bit system, depending on the system you are using.

  4. Select Manage this machine locally.
  5. Select the Recover option on the Acronis main window to open the Recover panel.


    The Recover data window is displayed.
  6. In the What to recover section, select Select data...
    The Data to Recover Selection window is displayed.
  7. In the Data to Recover Selection window, select Browse.
  8. In the Path field, enter the path where your backups are stored.
  9. Login as Administrator.
  10. Select the Archive file to recover and click OK.
    The Recover Data window is displayed.
  11. In the Where to recover section, select Required.
  12. In the Backup contents drop down menu, select Disks.
  13. Select the Disk 1 checkbox and select OK.



    14. The Recover Data window is displayed.
  14. Select OK to proceed with Disaster Recovery.

Ensure that only the keyboard and mouse are connected to the ports while doing a recovery.

To obtain the subnet mask and default gateway:

  1. From the Start Menu, select Run.
  2. Type in cmd.
  3. Type the ipconfig or ipconfig /all command.

For more information see, the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.
