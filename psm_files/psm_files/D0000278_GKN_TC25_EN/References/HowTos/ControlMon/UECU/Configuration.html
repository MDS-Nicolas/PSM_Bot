Reference – Configuration Files

This section contains information on the configuration files used by the Throttle System.

Only operators familiar with XML and the Throttle System configuration files should modify the configuration files. Errors in the configuration file will cause the system to behave unpredictably, or not run.

The Throttle System uses configuration files to determine the configuration of the system components. The configuration files are .xml files that are stored in the display computer in the throttle quadrant.

The block diagram below describes the location and content of the configuration files.



When the Throttle System's display computer starts, the display software automatically opens. You must close the display software to view the configuration files in the Windows 7 desktop.

MDS does not recommend using the display computer to edit the configuration files. Edit the configuration files using a standard desktop computer or laptop connected remotely.

Configuration Files

There are three configuration files:

  • Alarm Configuration File (alarmConfig.xml)
  • Channel Calibration Data Configuration File (calibrationData.xml)
  • Master Configuration File (ThrottleConfig.xml)

To Access the Configuration Files

  1. Start the Throttle System. For details, see Starting the UECU-500 Mark III Throttle.
    The display computer displays the Throttle System operator screens.
  2. In the top-left-hand corner of the screen, press the MDS Logo.
    The About window is displayed.


  3. Press the Exit button.
  4. Using Windows Explorer, navigate to the Config folder.

To Start the Display Computer GUI

  1. Press the UECU-500 shortcut .
    The display computer GUI starts.

Channel Calibration Data Configuration File (calibrationData.xml)

To prevent rendering the Throttle System inoperable, the Master Configuration file should only be modified by trained operators familiar with the Throttle System, the configuration files and XML.

The channel calibration data configuration file contains all of the calibration data for the channels that are able to be calibrated. The calibration is saved whenever a calibration is completed.

Do not modify this file manually! For details, see Calibrating a Channel. The only exception is when calibrating an analogue channel.

Master Configuration File (ThrottleConfig.xml)

In this example, the file name for the Master Configuration file is ThrottleConfig.xml. This name can be changed according to customer preference.

The master Configuration file contains the main configuration parameters for the Throttle System. Some of the parameters such as Actuator rigging data and Stop locations can be modified during normal operation.

To prevent rendering the Throttle System inoperable, the Master Configuration file should only be modified by trained operators familiar with the Throttle System, the configuration files and XML.

The Master Configuration file has the following four main elements which contain all of the other elements:

  • systemConfig - Contains system parameters
  • engineList - Contains engine parameters such as engine name, ID, defined Actuator/resolver/RVDT, Actuator rigging data, Trips and Stops
  • autoMode - Contains Auto Throttle Mode configuration data
  • calibrationData - Contains default channel configuration data

New calibration data is saved in the Master Configuration file in conjunction with the calibrationData.xml configuration file, but the Throttle System only reads calibration data from the calibrationData.xml configuration file.
