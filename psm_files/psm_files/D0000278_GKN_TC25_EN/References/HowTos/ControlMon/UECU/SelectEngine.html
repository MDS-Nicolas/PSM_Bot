How To – Select an Engine

This How To provides the steps required to select an engine.

To Select an Engine

Before the Throttle System can be used to test an engine it must be configured with the proper engine type. The engine can be selected by:

  • hardwired discrete inputs (Remotely Selecting an Engine)
  • proDAS engine configurations (Selecting an Engine Using proDAS)
  • the operator via the throttle quadrant (Selecting an Engine Using the throttle quadrant)

When an engine is selected, the throttle display software downloads the engine configuration parameters to the throttle controller. The parameters contain:

  • engine minimum and maximum ranges
  • number of levers and actuators
  • calibration curves
  • Auto Throttle configuration parameters
  • trip settings
  • stop positions (if configured)

If the engine selected contains any errors, a message is displayed to the user and the download is cancelled.

If the throttle controller detects that a new engine is requested via hardware discretes, it checks for interlocks to see if it is safe to transition to a new engine type. The following interlocks prevent transition:

  • Is the system in Run Mode?
  • Is fuel turned on?
  • Is a calibration in progress?
  • Is a rigging in progress?

If any of the above statements are true, the engine selection is not allowed to proceed until they are all false. Once the conditions are false, if the engine is still being requested to change, then the change continues. When the engine selection is allowed, the system changes to engine select mode and the engine selection screen is displayed. The engine list filters itself to only show the engine that is selected.

To accept the engine change request:

  • When the engine list updates and shows the selected engine, press the Download button.
    The engine is loaded from the master configuration file.
    • If there is an actuator defined and it needs to be rigged, the throttle quadrant automatically moves to the Rigging Mode screen
    • If the actuator does not need to be rigged or a resolver or RVDT is defined, the throttle quadrant automatically moves to the Run screen

Selecting an Engine Using the Throttle Quadrant

To select an engine:

  1. At the bottom of the display computer screen, press the Engine button.
    The Engine Selection screen is displayed.


    2. The Engine button is disabled if any of the following conditions are true:

    • Is the system in Run Mode?
    • Is fuel turned on?
    • Is a calibration in progress?
    • Is a rigging in progress?

  2. Select the desired engine from the list box. If required, press the Up or Down arrow to scroll the list.
  3. Press the Download button.
    The engine is loaded from the master configuration file and the configuration values in the Information about selected engine table updates:
    • If there is an actuator defined and it needs to be rigged, the throttle quadrant automatically moves to the Rigging Mode screen
    • If a resolver or RVDT is defined, the throttle quadrant automatically moves to the Run Mode screen
    • If there is no actuator, resolver or RVDT defined, an error is generated
