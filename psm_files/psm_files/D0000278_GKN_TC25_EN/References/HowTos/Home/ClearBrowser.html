How To – Clear your Browser History

This How To provides the steps required to clear the History from the Google Chrome browser.

If the PSM is being used in any other browser, please refer to your browser's help guide to properly clear your browsing history.

To Clear your Browser History

The Clear Browsing Data page can be displayed directly using the keyboard shortcut: CTRL+SHIFT+DEL.

  1. In the upper-right corner of the main browser window, select Settings (). The Settings menu is displayed.

    The History page can be displayed using the following keyboard shortcut: CTRL+H.

  2. Select History > History. The History page is displayed.
  3. From the left-side menu, select Clear Browsing Data. The Clear Browsing Data window is displayed.

    The Clear Browsing Data page can be accessed directly from the Settings menu by selecting More Tools > Clear Browsing Data.

  4. Ensure you have the following options selected:
    • Browsing History
    • Cookies and other site data
    • Cached images and files
  5. Ensure you have the time range set to All Time.
  6. Select Clear Data. The Clear Browsing Data window closes.
  7. In the upper left corner of the browser window, select Refresh ().
    All hyperlinks have now been returned to the active colour.
