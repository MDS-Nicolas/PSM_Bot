How To – Verify an Analogue Current Channel

This How To provides the steps required to verify an analogue current channel.

This How To applies to the following systems:

  • Fuel Measurement System
  • Temperature Measurement System

To prevent serious injury and/or irreparable damage to equipment, observe all manufacturer's warnings and safety precautions during operation and/or maintenance.

Electrostatic discharge can damage several components on an electronic module. To avoid damaging the module, wear an elctrostatic strap that is connected to a ground before working on a module.

To Verify an Analogue Current Channel

  1. Connect a variable DC voltage source (varies depending on signal type) to an analogue current channel input to simulate the applicable sensor output at the input of the VXI Technologies VT1503 SCP.
  2. Using the Sensor Calibration Application, perform a manual audit or a manual calibration as required. For details, see Calibrating Channels.
  3. Verify that the measured value displayed on proDAS corresponds to the values injected.
