How To – Check the Operation of the VXI DIO Card

This How To provides the steps required to verify the operation of the VXI digital input/output card.

To Check the VXI DIO Card

  1. Configure proDAS and start scanning. Use the How to Configure the VXI Chassis Using the Subsystem Editor procedure.
  2. Verify the operation of the E1458A 96-Channel Digital Input Module by connecting a jumper wire across the sensor input wires on the opto-isolator module. The LED below the module should turn on.

    3. If not, check the 5V and 24V power supplies to the board.

  3. Use the spotlight sticky or proDAS screen to check the status of the channels. Use the Creating Spotlight Sticky Windows procedure to create a spotlight sticky.
  4. Use a spotlight sticky to set the status of one or multiple output channels.
  5. Use a multimeter to measure the output at the opto-isolator board or at the end of the lines.
