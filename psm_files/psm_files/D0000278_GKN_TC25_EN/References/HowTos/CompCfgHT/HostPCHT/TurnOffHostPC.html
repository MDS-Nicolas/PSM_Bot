How To – Turn Off the Host Computer

This How To provides the steps required to turn off the Host Computer.

The Host Computer is turned off only for scheduled maintenance or if a power interruption is anticipated. The computer may be turned off from the Red Hat GUI or from the Linux Command Line.

Turning off the computer during shut down could damage the hard drive. Follow the shut down procedures to prevent loss of data or damage to components.

To Turn Off the Host Computer from the Red Hat GUI

  1. On the Host Computer, select the Actions menu from the Red Hat GUI.
  2. Select Log Out from the drop-down list of the Actions Menu. A log out window is displayed.
  3. Click Log Out Action in the log out window.
  4. Click OK in the log out window to log out of the system.
  5. Click Shut Down.
  6. Click Shut Down again. The system will automatically turn off.
  7. Turn off the Host Computer monitor.

To Turn Off the Host Computer from the Linux Command Line:

  1. On the Host Computer, in the Linux Command Line window, log in as root.
  2. Enter the following command: shutdown -h now.
     The system automatically turns off.
  3. Turn off the Host Computer monitor.
