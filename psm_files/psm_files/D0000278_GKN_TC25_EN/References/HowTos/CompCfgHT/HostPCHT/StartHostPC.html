How To – Turn On the Host Computer

This How To provides the steps required to turn on the Host Computer.

To prevent loss of data or damage to components, do not touch any buttons on the Host Computer front panel while the Host Computer is turning on. Do not reboot the system while the Host Computer is turning on.

To Start the Host Computer

  1. Turn on the monitor.
  2. On the front panel of the Host Computer, push the power button.
    The Host Computer boots up automatically and the Linux Operating System loads automatically.

    3. The proDAS database (Oracle and prodas_dbsrv; the communication program between proDAS and the database) also starts automatically.

  3. When the Host Computer has booted up and the Linux OS has been loaded, a Red Hat GUI is displayed on the Host Computer monitor.
