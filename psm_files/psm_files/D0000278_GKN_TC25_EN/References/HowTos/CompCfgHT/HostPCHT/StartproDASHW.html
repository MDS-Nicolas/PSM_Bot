How To – Turn On the proDAS Computers

This How To provides the steps required to turn on the proDAS computers.

The proDAS computers must be turned on in a specific order to ensure proper system operation.

To turn on the proDAS Computers

  1. In the proDAS equipment rack, power up the Host Computer. For details see Turning on the Host Computer.

    2. The Host Computer must be powered up before any other proDAS computer as the Host Computer acts as a server to the other computers and is a router between the Real-Time Network and the User Network.

  2. On the Console power up the Management Graphic User Interface (Mgt GUI) computer. For details, see How to Power Up the Computer.

    3. The Real-Time Display Computer maps drives from the Management GUI Computer, and the Management GUI Computer must be running windows before the proDAS Real-Time Display computers are powered up.

  3. On the Console, power up the proDAS Real-Time Display Computer(s). For details, see How to Power Up the Computer.
  4. On the Management GUI Computer, start the proDAS Management GUI Application. For details, see How to Start the proDAS Management GUI Application.
