How To – Change the User Mode

This How To provides the steps required to change the user mode to Multi-User Mode and Single User Mode.

To Change to Multi-User Mode

  1. Log in to the System Console as root.
  2. Enter the who command to see if other users are logged in to the operating system.
  3. Notify the other users that the system will be shutting down by entering the wall command, and then enter a notification message.
  4. Press Ctrl-d to send the message.
  5. Enter /sbin/telinit -t 600 5, where:
    • -t indicates the waiting duration before processing the command
    • 600 is the grace period in seconds to allow users to log out
    • 5 indicates multi-user mode with X11

    6. After the grace period elapses, the system displays a message to show the change to X11 multi-user mode. The Host Computer then changes to multi-user mode.

To Change to Single-User Mode

  1. Log in to the System Console as root.
  2. Enter the who command to see if other users are logged in to the operating system.
  3. Notify the other users that the system will be shutting down by entering the wall command, and then enter a notification message.
  4. Press Ctrl-d to send the message.
  5. Enter /sbin/telinit -t 600 1, where:
    • -t indicates the waiting duration before processing the command
    • 600 is the grace period in seconds to allow users to log out
    • 1 indicates single-user mode

    After the grace period elapses, the system displays a message to show the change to single-user mode. The Host Computer then changes to single-user mode.
