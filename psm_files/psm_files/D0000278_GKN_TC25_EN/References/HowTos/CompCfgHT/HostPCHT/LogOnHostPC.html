How To – Log In to the Host Computer

This How To provides the steps required to log in to the Host Computer.

To Log In to the Host Computer

  1. Turn on the Host Computer, if it has not already been turned on.
    A log in window is displayed.
  2. In the Username field, enter an applicable user name (technici, engineer, programm, or root).


    3. To log in as a superuser, type: root

  3. In the Password field, enter the user password.

    4. Note: If you have logged in as a superuser (root), type the root password.
