How To – Check the Disk Space on the Host Computer

This How To provides the steps required to check the disk space on the Host Computer.

To Check the Disk Space on the Host Computer

  1. Log in to the System Console as engineer or programm.
  2. At the prompt, enter df -k.
  3. Check the "% use" column.
    The number indicates the usage as a percentage (0% means the partition is empty and 100% means the partition is full). To display the amount of disk space used by each subdirectory and all directories within that subdirectory, enter du at the prompt .
  4. If there is not enough space in the partition to run the engine test, archive one or more of the earlier engine tests.
    For detailed information regarding archiving engine tests, see Test Archive Utility (TAU) under Standard Tools and Utilities in proDAS Integrated Help.
