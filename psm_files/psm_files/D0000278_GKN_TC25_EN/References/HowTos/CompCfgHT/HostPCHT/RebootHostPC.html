How To – Reboot the Host Computer

This How To provides the steps required to reboot the Host Computer.

To Reboot the Host Computer

The Host Computer must be rebooted after a maintenance operation or if the system crashes, freezes, or the System Console is not responding. Once the Host Computer has been rebooted, proDAS should resume normal operation. If proDAS does not resume normal operation, then proDAS must be restarted on the Host Computer.

  1. Log in to the System Console as root.
  2. Enter the who command to see if other users are logged in to the operating system.
  3. If other users are logged in to the operating system, enter the wall command, and then send a message to notify them of the impending shutdown and ask that they log out.
  4. Press Ctrl-d to send the message.
  5. When no other users are logged in, enter shutdown -r now.
    The Host Computer shuts down and reboots in multi-user mode.
