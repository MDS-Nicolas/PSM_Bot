How To – Power Down the Real-Time Display Computer from the Windows Taskbar

This How To provides the steps required to turn off the Real-Time Display Computer from the Windows Taskbar.

Power Down the Computer

Before turning off the computer, a proper shutdown using the Windows taskbar is required.

To Power Down the Real-Time Display Computer from the Windows Taskbar

To power down the Real-Time Display Computer, follow the procedure that best suits the situation as described below:

  • the proDAS software is configured and scanning or NOT scanning
  • the proDAS software is NOT configured and NOT scanning

The proDAS Software is Configured and Scanning or NOT Scanning

  1. From the Management GUI Computer, exit the proDAS Management GUI application by clicking Exit ProDAS.
  2. On the Real-Time Display Computer Windows taskbar, click Start, and then click Shut Down.
    The computer shuts down the Operating System and turns itself off.
  3. Turn off the Real-Time Display Computer monitor by pushing the power button.

The proDAS Software is NOT Configured and NOT Scanning

  1. On the Real-Time Display Computer Windows taskbar, click Start, and then click Shut Down.
    The computer shuts down the Operating System and turns itself off.
  2. Turn off the Real-Time Display Computer monitor by pushing the power button.
