How To – Perform a Crash Recovery on the Linux Host Computer

This How To provides the steps required to perform a crash recovery on the Linux Host Computer.

Perform a Crash Recovery on a Linux Host Computer

In the event that the local hard drive containing the operating system fails, use the following procedure to recover the data.

During a single session, you can restore several partitions or disks, one by one, by selecting one disk and setting its parameters first and then repeating these actions for every partition or disk to be restored.

  1. Power off the computer.
  2. Replace the damaged hard drive with a hard drive that has the same capacity as, or greater capacity than, the damaged hard drive.
  3. Power on the computer.
  4. Insert the Acronis Bootable Rescue Media (CD or DVD) into the Linux Host Computer and reboot the system.
  5. Select the Recovery task from the Acronis main window.
    This launches the Acronis Restore Data Wizard.
  6. Using the Acronis Restore Data Wizard GUI, navigate to the back-up NAS or USB directory.
  7. Select the most recent image file.

    Acronis reports an error if an incremental backup archive is selected but the corresponding full backup archive is no longer available. For example, once a new full backup is performed on Sunday, all the previous incremental backup archives become invalid.

  8. Select Restore Disk/partitions.
  9. Select the Disk to restore.
  10. Select a target Disk/partition to restore data to using one of the following locations:
    • its initial location
    • another disk/partition
    • an unallocated area

    When restoring an entire disk, the Acronis Restore Data Wizard analyses the target disk structure to verify the disk.

  11. If you have multiple restore tasks:
    • Select Yes, I want to restore another partition or hard disk drive
      The application returns to the disk/partition selection window.
    • Repeat steps 5 to 9 for each partition or disk drive
  12. Select the restore option:
    • use default options
    • set the options manually
    The Restoration Summary is displayed showing all selected options.
  13. In the Restoration Summary, choose one of the following options:
    • Click Back to make changes in the created restore task
    • Click Cancel to cancel the operation. No changes will be made to the disk(s)
    • Click Proceed to restore the disk/partition

    The disk/partition is restored.

For more information see, the Acronis True Image Echo Server for Linux User Guide.
