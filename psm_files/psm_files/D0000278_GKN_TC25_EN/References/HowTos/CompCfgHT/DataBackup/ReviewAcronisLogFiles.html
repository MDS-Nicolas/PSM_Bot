How To – Review Acronis Log Files

This How To provides the steps required to review the Acronis log files on any of the Windows based computers.

Log files are created automatically by the Acronis Data Backup & Recovery software every time a backup is attempted.

To Review the Acronis Log Files

  1. Double-click the Acronis icon located on the desktop.
  2. Select Log from the Navigation panel on the left side of the main Acronis window.
    The Log panel is displayed.
  3. From the Display menus, select the desired tasks and the period you want to review.
    The list of Acronis tasks for the date(s) selected is displayed.
  4. From the Log tab, select the desired log.
    The Log Entry Details window is displayed.
