How To – Recover Data from an Operating System or Hard Drive Failure on a Windows Based Computer

This How To provides the steps required to perform a crash recovery in the event that the computer's hard drive or Windows operating system fails.

To Recover the Data

  1. Power off the computer.
  2. Replace the damaged hard drive with a hard drive that has the same capacity as, or greater capacity than, the damaged hard drive.
  3. Power on the computer.

    4. Ensure that only the keyboard and mouse are connected to the computer while performing a recovery.

  4. Press F9 and verify that Boot from CD is the first item on the list.
  5. Insert the Acronis Bootable Rescue Media (CD or DVD) into the computer.

    6. In the event that an Acronis Bootable Rescue Media for Windows is not available or has been misplaced, the Acronis Bootable Rescue Media can be created using another proDAS Windows based computer which is running Acronis.

  6. Reboot the computer.
  7. A screen with three (3) options might be displayed. The options are:
    • continue Acronis with a 32 bit system
    • continue Acronis with a 64 bit system
    • continue booting the OS from the hard drive
    Select either 32 or 64 bit system, as required for the system configuration of the computer.
  8. From the Acronis Bootable Agent window, select Configure Network.
  9. Deselect the Auto Configuration check box.
  10. Specify the IP address previously assigned for the computer on the proDAS network. See To obtain the subnet mask and default gateway for more information.

    11. The IP address, Subnet Mask and Default Gateway information should be noted and saved to be used in the following step in the disaster recovery procedure.

  11. Specify the Subnet Mask and Default Gateway previously assigned to the computer.
  12. Click OK to close the Network Settings window.
  13. Select Manage this machine locally.
  14. Select the Recover option on the Acronis main window.


    The Recover data window is displayed.
  15. In the What to recover section, select Select data...
    The Data to Recover Selection window is displayed.
  16. In the Data to Recover Selection window, select Browse.
  17. In the Path field, enter \\<backupfoldername>\Backups
  18. Log in as Administrator.
  19. Select the Archive file to recover and click OK.
    The Recover Data window is displayed.
  20. In the Where to recover section, select Required.
  21. In the Backup contents drop down menu, select Disks.
  22. Select the Disk 1 checkbox and click OK.


    23. The Recover Data window is displayed.

  23. Click OK.
    Data recovery proceeds.
