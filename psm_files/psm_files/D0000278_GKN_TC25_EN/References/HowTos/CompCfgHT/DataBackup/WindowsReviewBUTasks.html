How To – Review Scheduled Tasks in the Acronis Backup & Recovery Application

This How To provides the steps required to review the scheduled tasks in the Acronis Backup & Recovery Application for a Windows based computer.

To review scheduled tasks:

  1. If it is not already started, start Acronis Backup and Recovery Advanced Workstation with Universal Restore from Windows.
  2. Select Backup Plans and Tasks from the Navigation panel on the left side of the main Acronis Backup & Recovery window.
    The list of scheduled backup plans and tasks is displayed.


  3. Use the panel buttons to view the details of the backup plans and tasks.

    4. The panel buttons are also used to create, run, edit, and delete backup plans and tasks.

    Data backup tasks as configured by MDS are designed for maximum protection against data loss. MDS highly recommends that the tasks NOT be changed.

For more information, see the Acronis Backup and Recovery Advanced Workstation with Universal Restore User Guide.
