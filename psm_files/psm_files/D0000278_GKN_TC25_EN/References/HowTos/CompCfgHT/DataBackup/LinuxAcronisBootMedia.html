How To – Create a Bootable Media for Acronis True Image Echo Server

This How To provides the steps required to create a bootable media for Acronis True Image Echo Server.

Create a Bootable Media for Acronis True Image Echo Server

This procedure uses the Acronis Media Builder wizard which creates a bootable media using the Acronis components that you select.

  1. Log in as the root user on the Linux Host Computer (RTE console).
  2. From the command prompt, type mediabuilder to launch the Acronis Media Builder application.
    The Acronis Media Builder application wizard starts.
  3. Click Next to continue.
  4. Select the Acronis True Image Echo Server full version.
  5. Click Next to continue.
  6. Click on the ISO image to select the ISO image file.

    7. Although there are different media types to choose from, MDS recommends you create an ISO file for your bootable media.

  7. Click Next to continue.
  8. Choose a location where the ISO file should be created:
    • Select: /disk3/users/engineer
    • File name: /disk3/users/engineer/xxxx.iso
  9. Click Next to continue.
  10. Click Proceed to proceed.
    A pop-up window indicates that the Bootable media has been created successfully.
  11. Click Ok to acknowledge the message.
    The ISO file (xxxx.iso) is created and saved in the engineer home directory.

For more information see, the Acronis True Image Echo Server for Linux User Guide.
