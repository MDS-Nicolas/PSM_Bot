How To – Cancel a Data Backup

This How To provides the steps required to cancel the data backup process on a Windows based computer.

Acronis Backup and Recovery Advanced Workstation with Universal Restore software is set up to create backup images of the local hard drive. The backup images are stored on a NAS.

The automatic backup tasks do not verify if proDAS is scanning before they begin. If testing is performed outside of regular work hours or on weekends, a backup may occur while proDAS is scanning. To ensure performance is not affected, the backup can be manually cancelled.

MDS does not recommend cancelling scheduled full or incremental backups because these processes have been staggered on a per computer basis and are expected to be completed quickly. Incremental backups are smaller and processed rapidly and are not expected to have any serious impact on computer performance.

To prevent serious damage to the hard drive, do not perform a hard reset or turn the power off during a backup. If required, you can shut down or restart the computer using the Start menu only.

A backup can only be cancelled when the user is logged in as an Administrator.

To Cancel a Backup Process

  1. Log in to the Management GUI PC as Administrator. See the Acronis Backup and Recovery User Guide.
  2. Select Backup plans and tasks from the Navigation panel on the left side of the Acronis Backup & Recovery window.
    The Backup plans and tasks panel is displayed in the main viewing area.


  3. Select a running backup task.

    4. Review the task details at the bottom of the screen to verify that this is the correct task to stop.

  4. In the Backup Plans and Tasks panel, click Stop.

Network Attached Storage (NAS)

The NAS is a network storage device with multiple hard drives. The hard drives are arranged as a mirrored RAID 1 disk set. The NAS is connected to the user switch.

For more information, see the Acronis Backup and Recovery User Guide.
