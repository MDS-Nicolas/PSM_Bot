How To – Mount a NAS (Network Attached Storage)

This How To provides the required steps to mount a NAS.

To Mount a NAS to the Linus Host Computer RTE System

  1. Log in as the root user on the Linux Host Computer.

    2. In the next step, replace XXXX with the system password. Replace yyyy with the system name.

  2. From the command prompt, enter: mount -t cifs -o username=engineer,password=XXXX //yyyy_nas/Backups /mnt/NASbackup

The NAS is mounted automatically during the RTE System boot-up.
