How To – Safely Mount and Disconnect a USB Drive on the Linux Host Computer

This How To provides the steps required to safely connect and disconnect a USB Drive to the Linux Host Computer (RTE System).

To Mount a USB Drive

  1. Log in as the root user on the Linux Host Computer (RTE System).
  2. From the command prompt, enter: mount /dev/sda1 /mnt/usbdrive

    3. The USB drive is mounted automatically during the Linux Host Computer (RTE System) boot-up.

To Disconnect a USB Drive

  1. Log in as the root user on the Linux Host Computer.
  2. From the command prompt, enter: umount /mnt/usbdrive
    The USB drive is disconnected.
