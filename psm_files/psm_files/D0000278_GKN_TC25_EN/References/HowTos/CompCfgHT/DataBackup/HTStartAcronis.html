How To – Start Acronis Backup and Recovery Advanced Workstation with Universal Restore from Windows

This How To provides the steps required to start Acronis Backup and Recovery Advanced Workstation with Universal Restore from a Windows based computer.

To start Acronis Backup and Recovery Advanced Workstation with Universal Restore

  1. Double-click the Acronis icon located on the desktop of the proDAS Management GUI Computer.
    The Acronis Backup and Recovery application starts.
  2. From the Acronis Backup and Recovery Management Console, select Connect to a management server.


  3. In the Management Server Connection window, enter the proDAS Management GUI PC name in the Machine field.


  4. Click OK.
  5. In the Access Credentials window, login as Administrator and enter a password for the Administrator’s account.


  6. Click OK.
