How To – Scan the Hard Drive for Errors

This How To provides the steps required to scan and verify the computer's hard drive. You can solve some computer problems by checking a drive or drives for errors. For example, you can check your computer's primary hard disk to fix some performance issues, or you can check an external hard drive if it is not working properly.

To Scan the Hard Drive for Errors

Electrostatic discharge can damage components and/or boards on your computer system. To avoid damage from electrostatic discharge when handling components and/or boards, wear an electrostatic strap that is connected to a proper ground.

Scanning the hard drive for errors may take several minutes to several hours, depending on the size of the hard drive.

  1. Stop or close all applications that are running on your computer.
  2. Select Start, Computer to display the installed Hard Disk Drives.
  3. Right-click the drive that you want to scan, and then click Properties.
  4. Click the Tools tab, and then, under Error-checking, click Check now. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
  5. There are two Check disk options. You can select one or both options as required:
    • To automatically repair problems with files and folders that the scan detects, select Automatically fix file system errors. If this is not selected, the disk check will report problems but not fix them.
    • To perform a thorough check, select Scan for and attempt recovery of bad sectors. This scan attempts to find and repair physical errors on the drive itself, but can take much longer to complete the process.
    • To check for both file errors and physical errors, select Automatically fix file system errors and Scan for and attempt recovery of bad sectors.
  6. Click Start.

    7. Depending on the size of your drive, this process might take several minutes. For best results, don't use your computer for any other tasks while it is checking for errors.

    If you select "Automatically fix file system errors" for a drive that is in use (for example, the partition that contains Windows), you will be prompted to reschedule the check for the next time you restart your computer.

Replace the hard drive if errors are detected by the disk scanning tool after repeated scans.
