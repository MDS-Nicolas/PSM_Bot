How To – Add a New User to Windows

This How To provides the steps required to add a user on a computer that is runnining a Windows operating system.

To Add a New User

Make sure that the computer and monitor are connected to a power source and that the associated keyboard, mouse and monitor cables are properly connected.

  1. Log on to the computer as Administrator.
  2. From the Start menu at the bottom of the screen, select Control Panel. The All Control Panel Items window is displayed.
  3. From the All Control Panel Items window select Administrative Tools.
  4. Select Computer Management.
    The Computer Management screen is displayed.


  5. Double-click on System Tools to expand the directory.
  6. Double-click on Local Users and Groupsto expand the directory.
  7. Double-click on the Users folder.
  8. From the Users panel, select More Actions -> New User...
    The New User screen is displayed.


  9. Enter the user name of the new user in the User name field (i.e. jsmith).
  10. Enter the name of the new user in the Full name field (i.e. Jane Smith).

    11. Entering the full name of the new user is optional.

  11. Enter a description of the new user in the Description field (e.g.: engineer).
  12. Enter the password for the new user in the Password and Confirm password fields.
  13. Select one of the following password options on the New User screen:
    • User must change password at next login
    • User Cannot Change Password
    • Password Never Expires
    • Account is disabled
  14. Select Create.
    The new user is added to the selected group.
