How To – Obtain the Subnet Mask and Default Gateway

This How To provides the required steps to obtain the subnet mask and default gateway on a Windows based computer.

To obtain the subnet mask and default gateway

Microsoft Windows has a console application called ipconfig that displays all current TCP/IP network configuration values.

  1. From the Start Menu, select Run. If Run is not available from the Start menu, type Run in the Type here to search field and press Enter.
    The Run window is displayed.
  2. In the Run window, in the Open field, type cmd and press Enter or click OK.
    The command window is displayed, showing the DOS prompt.
  3. To display the Windows IP configuration, type in ipconfig and press Enter.
  4. To display more details of the Windows IP configuration, type ipconfig /all and press Enter.
