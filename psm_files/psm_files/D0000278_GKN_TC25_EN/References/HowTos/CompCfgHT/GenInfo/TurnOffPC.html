How To – Turn Off a Windows Based Computer

This How To provides the steps required to turn off a computer that is running a Windows operating system.

Turning Off the Computer

Before turning off the computer, a proper shutdown using the Windows taskbar is required.

To Shut Down the Computer from the Windows Taskbar

  1. On the Windows taskbar, click Start, and then click Shut Down.
    The computer shuts down the Operating System and turns itself off.
  2. Turn off the computer monitor by pushing the power button.

To Shut Down the Computer using Ctrl-Alt-Delete

  1. Select and hold Ctrl-Alt-Delete. The Windows Security window is displayed.
  2. Select Shut Down... The Shut Down Windows window is displayed.
  3. Select Shut down from the drop-down menu, and then select OK
  4. The computer shuts down the Operating System and turns itself off.
  5. Turn off the computer monitor by pushing the power button.
