How To – Change a User Password in Windows

This How To provides the steps required to change a user password on a computer that is runnining a Windows operating system.

To Change a User Password

Make sure that the computer and monitor are connected to a power source and that the associated keyboard, mouse and monitor cables are properly connected.

  1. Log on to the computer as Administrator.
  2. Press Ctrl-Alt-Delete.
  3. Select Change a password... from the list of options.
    The Change Password window is displayed.

    4. The password for a different user can be changed by entering the user name and domain in the corresponding fields.

  4. Enter the current password in the Old password field.
  5. Enter a new password in the New password field.
  6. Enter the same new password in the Confirm new password field.
  7. Select OK on the Change Password screen.
    If the information entered in the Change Password window was entered correctly, a dialogue box advising that "Your password has been changed." is displayed.
  8. Select OK on the dialogue box.
