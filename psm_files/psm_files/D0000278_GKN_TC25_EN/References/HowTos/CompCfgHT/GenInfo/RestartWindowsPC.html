How To – Restart a Windows Based Computer

This How To provides the steps required to restart a computer that is running a Windows operating system.

Restarting the Computer from the Windows Taskbar

  1. On the Windows taskbar, click Start.
  2. Hover the mouse pointer over the Power Button , and select Restart from the pop-up menu.

    .

    The user is automatically logged off, any applications that are running are stopped and closed, the computer turns itself off and then turns itself back on.
  3. Once the computer has restarted, login as engineer.
