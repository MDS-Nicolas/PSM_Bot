Using the References

Overview

The References section contains the following on-line reference information:

  • an Acronyms and Abbreviations list containing the definitions for the acronyms and abbreviations used throughout the DocStation
  • a Glossary containing the definitions of terms that have specific meaning
  • Engineering Drawings that are grouped by system. Drawings can be opened by clicking on the hyperlinked sheet drawing name
  • Recommended Spare Parts List that provides a database of spare parts, part numbers, and suppliers for each subsystem
  • Vendor Documents (original manufacturer's manuals and data) that are grouped by subsystem
