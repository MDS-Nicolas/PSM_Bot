Recommended Spare Parts List

To reduce down time, it is recommended that a spare parts inventory be maintained for each subsystem. The Recommended Spare Parts List (RSPL) provides the following information:

  • manufacturer
  • quantity
  • part number
  • description
  • lead time

Where there is no MDS RSPL refer to the system or components original manufacturer vendor documentation.

The RSPL list can be viewed from this master list (arranged by subsystem) or from within the subsystem write-ups. The following is a hyperlinked list of systems containing RSPL's:
