IRIG-B

Overview

Inter-Range Instrumentation Group time codes, commonly known as IRIG time codes, are standard formats for transferring timing information. The system clocks of critical systems must be synchronised for consistent time stamping of data. High performance GPS receiver cards are used to acquire precise time from the GPS satellite system. The time is read from the GPS receiver card by the Host Computer over the PCI bus. The DDS system reads the IRIG-B signal over a co-ax cable directly from the receiver card. The other computers on the user network read the signal from the Host Computer over Ethernet using time-synchronising software.

The following systems use IRIG-B time stamping:

  • All proDAS computers
  • Dynamic Data System (DDS)
  • Other systems as required (Both MDS or client-supplied systems)
