Transient Log Definition Editor

Overview

The Transient Log Definition Editor is used to create and edit the definitions of a transient log. A transient log definition includes the set of channels to be logged, the logging rate, and the duration of the log.

The Transient Log Definition Editor provides the following functionality:

  • File
  • Edit
  • View
  • Configuration
  • Extras
  • Help
  • Index, Level, Enabled, Active, LastModDate, Name and Description
  • Creating a Transient Log Definition
  • Specific Properties of a Transient Log Definition
  • Import / Export a Tab-Delimited File

Creating a Transient Log Definition

To create a transient log:

The system checks every five minutes (default) whether there is enough disc space. If there is not, a warning is displayed in the UEL. This feature can be disabled in the proDAS initialisation file. If this feature is disabled, a warning is displayed only if there is less than 250 MB of free space when a transient log definition is created.

  1. From the Edit menu, select Append Data Record or, in the left pane of the splitter window, click the right mouse button and select Append Data Record from the context sensitive menu.
  2. Input the appropriate data in the Generic and Specific columns.

    When a new transient log definition is created the Duration field is intentionally initialised to an invalid value (i.e. 0.00002 mins.). The user must change the initial duration to a valid duration (up to 1440 mins.).

  3. Add channels to the log definition using the right pane of the splitter window.
    The right pane contains a list of all available channels in the configuration, as well as a list of all the channels currently selected for the log definition and the channel filters.
  4. To move one or more channels from the available list to the selected channels list:
    • double-click the channel or
    • select the channels using the -> and <- buttons or
    • to move all channels, use the -> all and <- all buttons

Specific Properties of a Transient Log Definition

The following specific properties of a transient log definition are displayed:

Column Name XML Attribute Name Description Type Acceptable Values
auto-report n/a Flag indicating whether the log is converted automatically, when stopped, into an ASCII format.

This flag is saved as string AUTO_REPORT in the description.		 Boolean true or false
Scan rate Rate The scan rate, in hertz, at which the log reads values. Float A positive number
Duration Duration The duration of the log in minutes. Float A positive number

Import / Export a Tab-Delimited File

The Transient Log Definition Editor allows Tab-Delimited transient log files to be imported and exported. The Tab-Delimited transient log files, which can be identified by the extension .def, comply with the structure detailed in the following table.

Row Description Type
1 Header or identifier. User Log
2 Information about the format version. 2.1
3 Description of the transient log. Description of atpTransLogDef2
4 Destination of import / export. File
5 Duration. 5000
6 Scan rate. 33.3
7 List of selected channels. fCalc2i, ...

When importing a transient log file, the file will be imported at the level where the Transient Log Editor is opened. For example, if the editor is opened to a level 6 configuration, the file will be imported at a level 6. If the editor is opened to a level 1 configuration, the file will be imported at a level 1. For more information on the levels, see Configuration Editors.

To import a Tab-Delimited file:

  1. Select Import Tab-delimited file... from the Extras menu.
    The Import window is displayed.
  2. Select the appropriate file from the Import window and select Open.
    Select Cancel to exit from the import operation.

    The name of the file must be the same as the subsystem name with which the file will be associated.

To export a Tab-Delimited file:

  1. Select Export Tab-delimited file... from the Extras menu.
    The Export window is displayed.
  2. Select the appropriate file from the Export window and select Save.
    Select Cancel to exit from the export operation.
