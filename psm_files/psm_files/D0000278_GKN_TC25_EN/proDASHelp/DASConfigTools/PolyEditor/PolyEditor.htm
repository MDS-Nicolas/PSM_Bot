Polynomial Editor

Overview

The Polynomial Editor is used to create and edit polynomials, defined by a sequence of float coefficients, to define a mathematical function.

Polynomials are most comonly used for unit conversions attached to hardware channels.


The Polynomial Editor provides the following functionality:

  • File
  • Edit
  • View
  • Configuration
  • Help
  • columns: Index, Level, Enabled, Active, LastModDate, Name, Description
  • Modification of the coefficients
  • Graphical display of a polynomial
  • Specific Properties of a Polynomial
  • List of Coefficients
  • Polynomial

Layout

It is possible to split the main window horizontally or vertically with an entry in the PolynomialEditor.ini file. When split vertically (see sample screen above) the "Selected Data Record" pane and the "Data Records" pane are parallel. When split horizontally (see sample screen below), the "Selected Data Record" pane and the "Data Records" pane are stacked.


The layout can be selected by setting the following ini properties:

[General]

# for horizontal layout:
WindowLayout=stacked

# for vertical layout:
WindowLayout=parallel

Specific Properties of a Polynomial

The specific properties of a polynomial are displayed in the table below.

Column Name XML Attribute Name Description Type Acceptable Values
Degree n/a Degree of the polynomial Integer read-only
Min Min Minimal x-value for which the polynomial is defined Float Smaller than maximum
Max Max Maximal x-value for which the polynomial is defined Float Greater than minimum
X Unit XEngineeringUnit Engineering unit for x values String A defined engineering unit
Y Unit YEngineeringUnit Engineering unit for y values String A defined engineering unit

List of Coefficients

The list of coefficients of the selected polynomial is displayed either in the right-hand pane or below the Data Records pane of the Polynomial Editor depending on the selected layout. For each coefficient, the list contains the following properties:

Column Name XML Attribute Name Description Type Acceptable Values
X Power n/a X power with which the coefficient is multiplied Integer read-only
Coefficient Value Value of the coefficient Float arbitrary

These coefficient values can be modified directly.

Polynomial

The Polynomial menu allows the user to edit polynomial coefficients and graph the results.

Modification of the Coefficients

The coefficients can be modified using the Polynomial menu.

The item Add Coefficient adds a coefficient at the bottom. As well, the user can position the cursor in the last row and select the down arrow button.

The item Delete Coefficient deletes a coefficient at the current cursor position by setting the value to zero. Leading zero coefficients are deleted.

Graphical Display of a Polynomial

The polynomials can be displayed graphically by selecting Display Polynomial from the Polynomial menu or by selecting Display Graphics.
