Engineering Unit Editor

Overview

The Engineering Unit Editor is used to create and edit the engineering units. The engineering units are not configuration specific, which allows all configurations to use a common EngineeringUnit.xml file stored in the proDAS\Data\Configuration directory. An engineering unit can be assigned to a channel using the Channel Editor.

Engineering units that are added and deleted are automatically synced in the .xml file, regardless of whether they are disabled or not.


The Engineering Unit Editor supplies the following functionality:

  • File
  • Edit
  • View
  • Tools
  • Help
  • Index, Enabled, Active, LastModDate, Name, and Description
  • Specific Properties of an Engineering Unit
  • Unit Category
  • Linear Transformation

Tools Menu

The Tools menu provides the following functionalities:

  • Convert
  • Invert

Convert

The Convert function is used to calculate the relationship between two units of the same category. Once a category is selected, the list of units in the sections "From" and "To" are displayed. The user can enter the "Input Quantity" value. "Reverse" is used to calculate the opposite relationship.

Invert

The Invert function inverts (calculates the offset and factor) the relationship between the current unit and the primary unit of the same category. This can be used to rectify a relationship entered in the wrong direction.

Unit Category

A unit category (e.g.: temperature, pressure) can be assigned to every engineering unit by using the Category column. To add a unit category that is not available in the drop-down list, type the name in the category field. The name entered in the category field is automatically added to the drop-down list.

The checkbox in the IsPrimary column can be used to set one engineering unit as primary unit for the category to which it belongs. Selecting more than one engineering unit as primary unit for a category is not possible. The Primary column displays the current primary unit of every engineering unit. Selecting another primary unit for a given category, the Factor and Offset are instantaneously calculated; the respectively columns and the relationship in the "Example" and "Relation to Primary Unit" columns are updated.

Linear Transformation

Engineering units for the same physical dimension can be converted into each other by linear transformations. These transformations are defined by the Factor and Offset columns. For instance, to define the linear transformation 1 °F = 1.8 * °C + 32, enter 1.8 in the Factor column and 32 in the Offset column. The measured values are converted without entering the specific formula.

The value of the Factor and Offset does not affect the channel value.

Examples

Temperature:

Primary unit: K

Current Unit: °C: Factor: 1, Offset: 273.15, Sample: 0 °C = 273.15 K, Relation: [K] = [°C] + 273.15

Current Unit: °F: Factor: 0.5555, Offset: 255.37, Sample: 0 °F = 255.37 K, Relation: [K] = 0.555*[°F] + 255.372

Time:


Primary unit: s [second]

Current Unit: m [minute]: Factor: 60, Offset: 0, Sample: 1 m = 60 s, Relation: [s] = 60*[m]

Explanation: A time interval of 1 minute takes 60 seconds, i.e. we must multiply the 1 by 60 to convert the minutes value into the seconds value.
