Onboard Maintenance System (Option*)

Overview

The Onboard Maintenance System (OMS) is part of the proDAS ARINC System used to test the FADEC on the Trent class engines.

Currently, the T700 and T800 OMS Graphical User Interfaces (GUIs) are available. The ARINC Display is also available for all ARINC equipped engines. Depending upon the selected engine configuration, the T700, T800, or ARINC Display can be displayed. The engine configuration is selected in the .config file located in users/RTE/bin/exe directory on the Host Computer.

The T800 OMS Initiated Tests are executed from the Management GUI Control Panel.

T700 OMS Window

To Start the T700 OMS Window:

  1. If configured, the T700 OMS Window starts automatically. It can also be started from a shortcut icon, only if selected in the configuration file.

  2. Select Lane A or Lane B to select the EEC channels and switch between each lane.
  3. Select Print to print the messages on the screen.
  4. Select Interactive OMS. The Interactive menu is displayed.

  5. Perform the test procedures as indicated in the display.
  6. Select Close to return to the Trent 700 Basic OMS Window.

T800 OMS Window

To start the T800 OMS Window:

  1. If configured, the T800 OMS Window starts automatically. It can also be started from a shortcut icon, only if selected in the configuration file.

  2. Select Get Fault Status to view current Fault Status (as shown in the status window).
  3. Select Increment Flight Leg to increment the Flight Leg number (e.g. displayed as Flight Leg: -126 at the upper right). A confirmation window is displayed prompting the user to select Yes or No to clear the message log.
  4. Select Shop Faults to view a list of shop faults. The Trent 800 OMS Shop Faults Index window is displayed.

  5. Highlight the desired Fault Page number and select Show Fault Data. The Trent 800 OMS Shop Fault Data window is displayed. This window provides detailed information of the fault.

  6. Select Print to produce a hardcopy of the fault information or Close to return to the Trent 800 OMS Shop Faults Index window.
  7. Select Prev Page or Next Page on the Trent 800 OMS Shop Fault Index window to view all of the listed Shop Faults.
  8. Select Print to produce a hard copy of the Shop Faults listed on the current screen.
  9. Select Close to close the Trent 800 OMS Shop Fault Index window.
  10. From the Trent 800 Basic OMS Window, select Equipment ID to view the Equipment Identification page. The Trent 800 OMS Equipment ID window is displayed. This window provides various detail about the equipment from the EEC, including its hardware, software and interface build standards.

  11. Select Print to produce a hard copy of the Trent 800 OMS Equipment ID window.
  12. Select Close to close the Trent 800 OMS Equipment ID window.
  13. Select Print from the Trent 800 Basic OMS window to produce a hard copy of the information.
  14. Select Clear from the Trent 800 Basic OMS window to reset the information window. A confirmation window is displayed prompting the user to select Yes or No to clear the message log.

T800 OMS Initiated Tests

The following T800 OMS initiated tests are available:

  • Variable Stator Vane Actuators (VSV)
  • Ignition System (IGN)
  • P2T2 Probe Heater (P2T2)
  • Air Oil Heat Exchanger Actuators (AOHE)
  • Fuel Metering Valve (FMV)
  • Thrust Reverse Interlock (TRI)

To start the OMS Window:

  1. From the Tools / Post-Test Function menu in Management GUI Control Panel, select Trent OMS.
    The Trent 800 OMS Initiated Tests window is displayed.

Variable Stator Vane Actuators (VSV)

To perform a Variable Stator Vane Actuators (VSV) test:

  1. Select VSV. The Variable Stator Vane Actuator window is displayed.
  2. Follow the instructions listed in the dialogue box and select Continue. The EEC will respond with a Ready dialogue box when the test can be continued further.
  3. Select Continue to display the next instructional dialogue box.
  4. To cancel a test at any point in the procedure, select Abort.
  5. When the test has been successfully completed the EEC will respond with a Pass or Fail message.
  6. If testing fails, a dialogue box is displayed.
  7. Select Continue to return to the Trent 800 OMS Initiated Test window.
  8. If testing is inhibited, a dialogue box is displayed, giving a reason for the inhibited test.
  9. Select Retry to return to the instruction dialogue box or Abort to return to the Trent 800 OMS Initiated Test window.
  10. When testing is complete select Close on the Trent 800 OMS Initiated Test window. The Management GUI Control Panel is displayed.
