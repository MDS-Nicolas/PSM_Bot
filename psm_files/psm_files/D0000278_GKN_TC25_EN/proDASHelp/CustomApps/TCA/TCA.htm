Thrust Calibration Application (TCA) (Option*)

Overview

The Thrust Calibration Application (TCA) is a component of the Thrust Calibration and Thrust Measurement System. The Thrust Calibration System applies variable force to the thrust frame. The Thrust Measurement System acquires data from the force measurement instrumentation on the thrust frame and transmits the data through proDAS to the TCA. Load data may be acquired automatically or manually through preset thrust load points.

A manual audit has a greater flexibility to acquire additional data around specific thrust points.

The Thrust Calibration PLC receives target load commands from the TCA. The PLC transmits control signals to the hydraulic components (valves, hydraulic cylinders and the Servo Control Module) in the test bed, and receives feedback data from measurement components (load cells and associated signal conditioners).

Thrust Measurement Hydraulics apply increasing and then decreasing loads to the live (movable) portion of the thrust frame with hydraulic cylinders. The PLC maintains the target load and transmits feedback data to the TCA.

The TCA performs the following specific functions:

  • user log on
  • start-up verification
  • zeroing thrust measurement equipment
  • exercise the thrust frame
  • verify load cell preloads
  • manual auditing
  • automatic auditing
  • viewing previous audit reports
  • creating new thrust audit and master calibration reports

The interface to the TCA consists of the Audit/Master Calibration Page and Pre-Audit Page GUIs.

The TCA may be launched:

  • as a stand-alone application where the username and password are not automatically passed
  • from proDAS through the Management GUI Control Panel, where the username and password are automatically passed

There are five security levels associated to each user. Access to the TCA functions is limited by the security level (functions that are not listed in the table have no security level):

TCA Function
modify target tolerance and signal definition limits
zero working thrust measurement
accept single point master calibration
exercise thrust frame
perform automatic audit or master calibration
perform manual audit or master calibration

A level five user may execute any function and modify the security level associated to each TCA function above.

The displayed date format (dd/mm/yy or yy/mm/dd) in the application is based upon the regional settings of the Management Interface Computer.

Starting the TCA

The TCA can be started either as a stand-alone application or from the Management GUI Control Panel.

To start the TCA as a stand-alone application:

  1. Double-click the TCA executable file from the Management Interface Computer directory structure.
    The Login window is displayed.
  2. Enter a username in the Username field.
  3. Enter a password in the Password field.
  4. Select Login.
  5. If the Test Bed is configured for both forward and reverse thrust calibration, the Calibration Direction Prompt is displayed.

To start the TCA from the Management GUI Control Panel:

  1. From the Management GUI Control Panel (below), select Tools / Pre-Test Function / Thrust Calibration.
    The Calibration Direction Prompt is displayed.
  2. Select Forward or Reverse. The Calibration Mode Prompt is displayed.
  3. Select Manual or Automatic. If Manual is selected, the Pre-Audit Page is displayed. If Automatic is selected, the Start or Quit HPU Prompt (below) is displayed.
  4. Select Start HPU to continue or Quit to exit TCA. If Start HPU is selected, the Start HPU prompt is displayed.
  5. Select Yes to start the Hydraulic Power Unit (HPU) or No to exit TCA. If Yes is selected, the HPU is started and the Exercise Thrust Frame Prompt is displayed.
  6. Select Yes to exercise the thrust frame or No to skip the exercise. The Select Configuration Information window is displayed.
  7. Select the required Engine Type, Intake Type and Shutter Position from the drop-down lists on the Select Configuration Information window. Select OK. The Audit/Master Calibration Page is displayed.

  • displaying audit data in numerical or graphical form
  • displaying audit data reports

Manual Audit

Starting a Manual Audit

The Pre-Audit Page is displayed before the manual audit is started to allow previous calibration reports to be reviewed, the thrust frame to be exercised, and the offset on the test bed load cells to be calibrated or "zeroed".

  1. Select Manual from the Calibration Mode Prompt during TCA log on:
    The Pre-Audit Page is displayed. Manual Audit and Thrust mode (Forward or Reverse) which are selected during TCA log on are displayed in fields at the top of the screen. Manual Audit mode is also indicated in the Status field. The Review Calibration Histories, Start HPU and Exit buttons are enabled.
  2. Select Start HPU to start the Hydraulic Power Unit (HPU). A Warning Prompt is displayed.
  3. Select Yes to start the HPU or No to abort HPU startup. If Yes is selected, the Exercise, Begin Audit and Zero Working buttons are enabled.

Review Calibration Histories

  1. Select Review Calibration Histories. The TCA\Reports folder is displayed in a window.
  2. Select an audit report .xls (spreadsheet) file for review if required.

Exercise the Thrust Frame

  1. Select Exercise. The Exercise Prompt is displayed.
  2. Select Yes to exercise the thrust frame or No to abort the exercise. If Yes is selected, the live portion of the thrust frame is moved to a peak value and back to zero three times by the Thrust Hydraulics.

    The Exercise button is enabled at a higher user security level.

Zero Working Load Cells

Select Zero Working. The offsets on the working load cells in the test bed are "zeroed".

The Zero Working button is enabled at a higher user security level.

Performing a Manual Audit

  1. Select Begin Audit. The Begin Prompt is displayed.
  2. Select Yes to begin the audit or No to abort the audit. If Yes is selected, the Select Configuration window is displayed.
  3. Select the appropriate Engine Type, Intake Type and Shutter Position data from the drop-down lists in the window.
  4. Select OK. The offsets in the Transfer Standard load cells are automatically "zeroed" during the transition from the Pre-Audit Page to the Audit/Master Calibration Page. An Error Prompt is displayed if the resulting calibration value FnTS (Force - Transfer Standard) exceeds an allowable limit.
    Select Yes to abort the calibration audit. The Pre-Audit Page is displayed.
    Select No to continue the calibration audit. The Audit/Master Calibration Page is displayed. The Accept, Save Comment and Cancel buttons are enabled. If the thrust frame is preloaded, the working Load Cell preloads are also checked and a warning prompt is displayed if preloads are not within limits.

    Select Cancel to abort the audit and return to the Pre-Audit Page anytime.

    The standard deviation limit is preset, and can only be modified at a higher user security level.

    The target load tolerance is preset, and can only be modified at a higher user security level. Target loads are modified only under special circumstances, such as troubleshooting.

    The target load automatically changes to the next default level (10,000, 20,000, 40,000 etc.).

  5. Select Apply Load. The Thrust Hydraulics apply the target load to the thrust frame. The Status field displays a "Driving to target value" message until the target load has been applied to the frame. The Accept and Apply Load buttons are disabled during this time. The Status field displays a "PLC is at target" message and Accept is enabled when the transfer standard is within the tolerance of the target load and the working reading is within standard deviation limits.
  6. Select Accept to record the thrust data in the Current Data Bank. The acquired data (target (lbf), FnTS (lbf), FnMeas (lbf), FnCorr, error and temperature) is displayed in the Incrementing column of the Current Data Bank field. The background colour of the row is red if the Error has exceeded the allowable maximum. Select Undo to delete the recorded data from the last target point.

    The Start Decrementing button is enabled after a single non-zero load point has been recorded. If Start Decrementing is selected, a Start Decrementing Prompt is displayed (see Step 8.)

    A Time Out Prompt is displayed if the duration for acquiring the target load data is exceeded.
    Select Continue to proceed with the "wait operation" or Cancel to abort the calibration and display the Pre-Audit Page.
  7. Repeat Steps 5 - 6 for the required incrementing target loads. Additional target load data may be acquired beyond the maximum default point. If Apply Load is selected for target loads beyond the maximum default level, a Start Decrementing Prompt is displayed.
  8. Select Yes to decrement the target load. Select No to abort decrementing the target load.
  9. If Yes is selected, the decrementing targets are set to the same level as the incrementing targets. The targets cannot be altered.
  10. Select Apply Load. The Thrust Hydraulics apply the target load to the thrust frame. The Status field displays a "Driving to target value" message until the target load has been applied to the frame. The Accept and Apply Load buttons are disabled during this time. The Status field displays a "PLC is at target" message and Accept is enabled when the transfer standard is within the tolerance of the target load and the working reading is within standard deviation limits.
  11. Select Accept to record measured load data in the Current Data Bank. The acquired data (target (lbf), FnTS (lbf), FnMeas (lbf), FnCorr, error and temperature) is displayed in the Decrementing column of the Current Data Bank field. The hysteresis between the incrementing reading and the decrementing reading is calculated an displayed. The background colour of the row is red if the Error has exceeded the allowable maximum. Select Undo to delete the recorded data from the last target point.

    A Time Out Prompt is displayed if the duration for acquiring the target load data is exceeded.
    Select Continue to proceed with the "wait" or Cancel to abort the calibration and display the Pre-Audit Page.

  12. Repeat Steps 9 - 12 for the required decrementing target loads. A Graph Prompt is displayed at the minimum default target load level.
  13. Select Yes to display the Current Data Bank in graphical form; proceed to Step 16. Select No to bypass graphing and continue.
  14. If No is selected and the audit failed, a Bank Data Prompt is displayed. If the audit passed, there is no option to bank the audit.

    The TCA does not offer the option to bank an audit towards the next calibration if the audit passed. If the audit is successful, there will be an option to save the audit and extend the calibration due date.

    If No is selected, a Rejected Audit report is generated and the Audit/Master Calibration Page is displayed. If Yes is selected, the calibration audit data is entered into the Banked 1 field and a Warning Prompt is displayed.
    Enter comments into the Comments field.

  15. Select OK. A graph of the current data bank is displayed. The Audit Pass/Fail field indicates a Pass or Fail condition for the audit.
    • select Current, Banked 1, Banked 2 or Banked 3 to display banked data in graphical form
    • select Wrk vs. App (working thrust vs. applied thrust) or Error vs. App (measurement error vs. applied thrust) to change the type of graph to display ( Wrk vs. App is shown above)
    • select Show Labels to display the thrust measurements values above the thrust points (above, in illustration)
    • the area of the graph may be increased ("zoomed") by defining a zoom area with the cursor
    • select Zoom Out to return the graph to its original resolution

  16. Enter comments into the Comments field and select Save Comment.
  17. Select X to exit the graphical display. A Bank Data Prompt is displayed.
  18. Select Yes to store the current data under the Banked 1 tab. The No. Banked Audits field is incremented to 1.
    Select No to delete the data and display the Audit/Master Calibration Page. Do Next Audit, Update Master Calibration Data and Accept Audit New Cal Due Date are enabled.
  19. Select Do Next Audit to record another set of audit data. Select Update Master Calibration Data to create a new Master Calibration. Select Accept Audit New Cal Due Date to update the calibration due date(if the audit was successful).

Accept Audit New Calibration Due Date

The Working Thrust Calibration Due Date may be updated (moved forward 30 days) after a successful audit, i.e., every data point of the audit has error and hysteresis below the configurable maximum, and Pass is displayed in the Audit Pass/Fail field. However, a bank of failed data may be used to update the Calibration Due Date at a higher user security level.

  1. After banking audit data, select Accept Audit New Cal Due Date. The Update Due Date calendar is displayed.
  2. Select the calibration day with the mouse cursor and button
  3. Select the calibration month with the arrow buttons on the calendar
  4. Select Update. The calibration due date is updated.

Creating a New Master Calibration

A new Master Calibration can be created (updated) if three consecutive sets of banked data pass the consistency check, and the new calibration BPT is within the configurable tolerance of the Bed Characteristic. The Master Calibration Consistency Pass/Fail field displays Pass to indicate three sets of good data. However, even a single set of audit data banked may be used to update the Master Calibration data at a higher user security level. If a user has logged at an authorization level that is lower, a prompt for a higher user level is displayed.

A comment is added to the Master Calibration report which states that a single point Master Calibration has been authorized.

A new Master Calibration cannot be created that contains data which falls outside the tolerance for the Bed Characteristic Line. The line represents the original Master Calibration BPT for the bed.

  1. Select Update Master Calibration Data to create a new Master Calibration. A Confirmation Prompt is displayed.
  2. Select Yes to create a new Master Calibration. Select No to return to the Audit/Master Calibration Page. If Yes is selected, an Update Due Date calendar is displayed.
    • select the calibration day with the mouse cursor and button
    • select the calibration month with the arrow buttons on the calendar

  3. Select Update. A new Master Calibration is created and the calibration due date is updated.

The View Report and Print Report buttons are disabled until the Update Master Calibration button is selected.

Automatic Calibration Audit

Starting an Automatic Audit

An automatic calibration audit is performed with a preset schedule of events and target loads. The audit:

  • exercises the thrust frame
  • zeros the load cells
  • applies a load by ramping thrust up and down through preset target points
  • creates a Master Calibration if required
  • generates an audit report
  1. Select Automatic from the Calibration Mode Prompt during TCA log on.
    A Start or Quit HPU Prompt is displayed.
    Select Start HPU to start the Hydraulic Power Unit. Select Quit to exit TCA.

  2. Select Start HPU to start the Hydraulic Power Unit (HPU). A Warning Prompt is displayed.
  3. Select Yes to start the HPU or No to abort HPU startup. If Yes is selected, the Pre-Audit Page and the Exercise Prompt is displayed.
  4. Select Yes to exercise the thrust frame or No to abort the exercise. If Yes is selected, the live portion of the thrust frame is moved forward and backward by the Thrust Hydraulics. The Select Configuration window is displayed.
  5. Select the appropriate Engine Type, Intake Type and Shutter Position data from the drop-down lists in the window.
  6. Select OK. The offsets in the Transfer Standard load cells are automatically calibrated or "zeroed". An Error Prompt is displayed if the calibration value exceeds an allowable limit.
    If the thrust frame is pre-loaded, the working Load Cell preloads are also checked and a warning prompt is displayed if preloads are not within limits.

  7. Select Yes to abort the calibration audit and display the Pre-Audit Page. Select No to continue the calibration audit and display the Audit/Master Calibration Page. The Status field displays "Start Automatically Recording Data". The audit automatically acquires target load point data.

    Select Cancel to abort the audit and return to the Pre-Audit Page anytime.

  8. The Current data bank of the Audit Master Calibration Page is populated with data during the audit.
    The Graph, View Report and Print Report buttons are enabled when the audit is completed. If the audit data is within target load tolerance when the audit is completed:
    • A Pass Prompt is displayed
    • the Accept Audit New Cal Due Date button is enabled
    • Pass is displayed in the Audit Pass/Fail field
    If the audit data is outside of target load tolerance when the audit is completed:
    • Fail is displayed in the Audit Pass/Fail field
    • no Pass Prompt is displayed
    • the Do Next Audit button is enabled
    • the Accept Audit New Cal Due Date button is not enabled
    • the Update Master Calibration button is enabled
    • a Graph Prompt is displayed

  9. Select OK if the Pass Prompt is displayed. A Graph Prompt is displayed.
  10. Select Yes to display the Current Data Bank in graphical form. Select No to bank the calibration audit data and enable Do Next Audit (this option is only available if the audit failed). If No is selected, a Bank Data Prompt is displayed.

    The TCA does not offer the option to bank an audit towards the next calibration if the audit passed. If the audit is successful, there will be an option to save the audit and extend the calibration due date.

    If No is selected, the Audit/Master Calibration Page is displayed. If Yes is selected, the calibration audit data is entered into the Banked 1 field.

  11. If Yes is selected from the Graph Prompt, a warning Prompt is displayed.
  12. Select OK. A graph of the current data bank is displayed.
    • select Current, Banked 1, Banked 2 or Banked 3 to display banked data in graphical form
    • select Wrk vs. App (working thrust vs. applied thrust) or Error vs. App (measurement error vs. applied thrust) to change the type of graph to display ( Wrk vs. App is shown above)
    • select Show Labels to display the thrust measurement values above the thrust points (above, in illustration)
    • the area of the graph may be increased ("zoomed") by defining a zoom area with the cursor
    • select Zoom Out to return the graph to its original resolution
      Enter comments into the Comments field and select Save Comment.

  13. Select X to exit the graphical display. A Bank Data Prompt is displayed.
  14. Select Yes to store the current data under the Banked 1 tab. The No. Banked Audits field is incremented to 1.
    Select No to delete the data and display the Audit/Master Calibration Page.
  15. Select Do Next Audit to record another set of audit data. Repeat steps 8 - 12.
  16. Select Update Master Calibration Data to create a new Master Calibration.
  17. Select Accept Audit New Cal Due Date to update the calibration due date.

View Audit Reports

An audit report is generated after current audit data is banked, when an automatic or manual audit is completed and the View Report button is enabled. The report is generated in .xls (spreadsheet) form and consists of four sections:

  • Audit Report
  • Current Data
  • Banked Data
  • Master Calibration Data
