ARINC Channel List (Option*)

Overview

The ARINC Display shows the ARINC information being transmitted to and received from the EEC (Electronic Engine Control) and provides controls for saving and printing the ARINC information.

If a customer has the ARINC Subsystem the start scan script will be set up to automatically start the ARINC Display.

The following information is displayed for each channel:

An icon in the Type column of the ARINC Display identifies the CHANNEL data type (i.e.: Parameter Data, Maintenance and Status Data, and Transmitted Data).

  • Data Type
    • Parameter Data (analogue received data)
    • Maintenance and Status Data (discrete received data)
    • Transmitted Data
    • All Data
  • Channel Name
  • Sign/Status Matrix (SSM)
  • Transmit Receive Indication (TRI)
  • Message

The channel list can be sorted by type or channel name and in ascending or descending order by clicking on the column header (i.e.: Type or Channel Name) or selecting a sort option from the View menu. A small triangle indicates the selected column (i.e.: Type or Channel Name) and the sort order (i.e.: ascending or descending). To toggle the sort order, click on the column header with the triangle.

ARINC Display
Buttons Panels and Fields Function
The File menu and the application tool bar provide options for saving the ARINC Channel List or printing the ARINC Channel List to a file.
The Edit menu options allow the operator to search for a channel.

If Find... is selected, the Find dialogue box is displayed.

Select Find Next to search forward, from the currently selected channel, to find a channel that matches the last specified search criteria.

Select Find Previous to search backward, from the currently selected channel, to find a channel that matches the last specified search criteria.
The Font dialogue box is displayed if Font Settings is selected from the View menu.
The Find dialogue box is displayed if Find... is selected from the Edit menu.

The Find dialogue box search options allow the user to:
  • specify an ARINC Label
  • specify a Channel Name
  • set the search direction (i.e.: Next or Previous)

The ARINC Label and Channel Name are case sensitive.

Only the currently displayed filtered list is searched, not the entire ARINC channel list.

If both the Channel Name and the Label are specified, a logical OR is used to find the matching channels.

After the search has been completed, the number of occurrences is indicated in the status bar and the first matching channel is marked as selected in the channel list.
The View menu options allow the user to:
  • change font settings
  • filter the channel list to include:
    • all lists
    • parameter data
    • maintenance and status data
    • transmitted data
  • sort the channel list by:
    • type
    • channel name
    • ascending order
    • descending order
  • adjust the column settings
  • show or hide the Toolbar for the application
  • show or hide the Status Bar
If Column Settings... is selected from the View menu, the Column Settings dialogue box is displayed.
The Column Settings dialogue box is displayed if Column Settings ... is selected from the View menu.

The Column Settings dialogue box options allow the user to:
  • show or hide any column
  • change the order of the columns displayed in the channel list with the up and down arrows
From the Help menu, the user can:
  • access the first page of the proDAS Integrated Help (i.e.: select Getting Started)
  • access all specific help topics referring to the ARINC Display (i.e.: select Topic)
  • displays the application information,comprising the name, version number, and copyright
    (i.e.: select About ARINC Display)
Select the Save button to save the Channel List to a file. The user has the option of defining the file location.
Select the Print button to print the Channel List using the Continuous Print Server.
Select the Find button to search for a channel by channel name and/or label.
Select the Find Next button to search forward, from the currently selected channel, to find a channel that matches the last specified search criteria.
Select the Find Previous button to search backward, from the currently selected channel, to find a channel that matches the last specified search criteria.
Select the View All Lists button to view all the channel lists.
Select the View Parameter Data button to filter the channel list and view only the parameter data (i.e.: analogue data received from the EEC).
Select the View Maintenance and Status Data button to filter the channel list and view only the maintenance data (i.e.: discrete data received from the EEC).
Select the View Transmitted Data button to filter the channel list and view only the data transmitted to the EEC.
