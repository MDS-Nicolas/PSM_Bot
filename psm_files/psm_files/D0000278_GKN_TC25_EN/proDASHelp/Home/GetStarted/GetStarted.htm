Getting Started

Overview

The proDAS Integrated Help is a web based user friendly support package offering concise and informative data supported by graphical illustrations and photographic images. Coupled with the mouse point and click function, the user is capable of navigating the entire proDAS Integrated Help system. Specific text items may be linked to other areas of this document ( Hyperlinks ), enabling the reader to jump directly to another source of information that best describes or illustrates specific information about the MDS delivered products.

This versatile integrated web-based utility comes with dynamic tools such as; the Adobe Acrobat Reader that enables the user to open, navigate and print pdf files; JavaScript and Active-X plug ins, are software modules that add specific features to a system allowing access to specific file formats e.g. various graphics and 3D file formats. The standard resolution size for the proDAS Integrated Help is 1280x1024, with the viewing screen area adapting to the various monitor resolution sizes. For smaller viewing areas, scroll bars are automatically placed along the bottom (horizontal left and right scrolling) and the right side of the screen (vertical upward and downward scrolling). The proDAS Integrated Help creates a unique, user friendly learning environment for all levels of users.

Hyperlinks

Hyperlinks, denoted by coloured underlined text ( Hyperlinks ), are text items that are linked to another page or paragraph that directly relates and supports the hyperlinked item. The Hyperlink allows for rapid transport from one location in the MDS Product Support Manual to another. By placing the mouse cursor over a hyperlink, the cursor image changes from an arrow to a white hand with a pointing finger. Clicking the cursor while over a hyperlink opens the page containing the information indicated by the hyperlink in the main viewing area. All topics listed in each information module are linked to the related subject matter. Click on any hyperlinked topic to display the subject in the main viewing area.

Hyperlinks found within the system write-ups display related information. The link may be specific (i.e. Click here to...) or implied and embedded in the main text. In either case, click on the link to display the associated information in the main window.

It is recommended that your browser history be cleared regularly to reset all links back to the active colour. See How To - Clear your Browser History.

Using the proDAS Integrated Help

The proDAS Integrated Help is the user's primary source of reference information for the MDS delivered proDAS system. A user friendly NavMenu is always available at the left side of the viewer's screen. The user is able to quickly navigate via Hyperlinks (to jump) from one information module to another or, where applicable, to easily travel within the currently opened information module. The NavMenu is equipped with clearly marked main categories for quick navigation through the proDAS Integrated Help. A category represents a group of technical or non-technical information modules.

When a category has been selected, the respective group of topics is displayed under the category. When a topic has been selected (user has clicked on the topic text), the subject matter list is displayed. When selected, the subject matter is displayed in the main viewing area.

The NavMenu is comprised of categories. These categories may be technical or non-technical. Each category contains topics. Each topic contains subject matter related specifically to that topic. The NavMenu expands and contracts (like an accordion) to display more or less infomation on the selected category or topic. The NavMenu has a dark grey background colour and white text.

NavMenu Main Categories

The NavMenu for proDAS Integrated Help is divided into different categories. Each category contains a group of related information modules that contain topic indexes to navigate directly to the detailed subject matter. The NavMenu displays the following main categories:

  • Home
  • Testing with proDAS
  • proDAS Configuration Tools
  • Custom Applications
  • proDAS Subsystem Configuration
  • Standard Tools and Utilities
  • References
  • Search

Home

The Home Page is the first page displayed when the proDAS Integrated Help is initiated. The Home Page is similar to the title page in a book, displaying the Getting Started module. The Getting Started module is a user friendly introduction to the definitions and operation of the proDAS Integrated Help.

Testing with proDAS

The Testing with proDAS category focuses on the day-to-day operational use of proDAS and is targeted at the Operators using the proDAS system. The Testing with proDAS category contains supportive information such as an introduction to proDAS, proDAS features on operation, running an engine test, and viewing the engine test data.

proDAS Configuration Tools

The Configuration Tools category includes information on the Configuration Tools such as Configuration Editors, Macro Editor, Channel Editor, etc.

Custom Applications

The Custom Applications category includes custom applications such as Engine Trim Balance, TOCEUM, EASIFIT, etc. that are required when working with specific equipment.

Option*: Denotes that this proDAS feature is not part of a standard installation. Please refer to the "Home" section of the client specific Product Support Manual (PSM) for details on the current system configuration.

proDAS Subsystem Configuration

The proDAS Subsystem Configuration category includes technical and informative modules dealing with the configuration of the proDAS subsystems. The proDAS Subsystem Configuration category discusses the software systems and their relationships and is targeted at the individuals who are responsible for preparing proDAS configurations to run an engine test.

Option*: Denotes that this proDAS feature is not part of a standard installation. Please refer to the "Home" section of the client specific Product Support Manual (PSM) for details on the current system configuration.

Standard Tools and Utilities

The Standard Tools and Utilities category encompasses all of the software tools and utilities necessary to modify and maintain proDAS. It is also an extension of the information presented in Testing with proDAS and proDAS Configuration. These software tools and utilities include, but are not limited to, Calibrating Channels, Setup Data Recording, Replay Control, Building Reports with RAVE, User Security System, etc.

References

A series of on-line reference documents is included in the proDAS Integrated Help to further guide and inform the user. These reference documents include, but are not limited to, the following:

  • Interface Control Documents
  • RTE and GUI Error Codes
  • Trace Utility
  • Software Initialisation Files (samples)

Search

The proDAS Integrated Help contains a user friendly search feature that is accessible from the NavMenu. When the feature is selected, a customised search page is displayed. Search criteria (words or phrases) are entered into the page, and the feature displays information that is found within MDS Product Support Manual which contains the criteria.
