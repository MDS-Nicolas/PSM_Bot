Online Help Tools (OHT)

Overview

The content of the Online Help Tools (OHT) is created and maintained by the users of the system and is not a part of proDAS Integrated Help.

The Online Help Tools (OHT) is an ActiveX computer program controller plug-in within the proDAS Integrated Help. OHT allows users to view RTF files with user-defined content, with some authorized users having the ability to edit the content, if needed. The OHT is embedded into the current version of proDAS Integrated Help. When a user selects a help topic in the proDAS application, the OHT loads the correct page.

The RTF files are stored in the following directory \\rtehost\rte\proDASHelp\UserHelpFiles.

Any RTF file created and placed in the above directory is automatically displayed in the User Defined Help drop-down menu.

Using the Online Help Tools (OHT)

To use the Online Help Tools (OHT):

  1. From any proDAS Application (i.e. Sensor Editor), select Help / Topics / Online Help Tools.
    The User Defined Help window is displayed.
  2. From the drop-down menu, select the desired topic (i.e.: Introduction).
  3. Select to open the selected topic (i.e.: Introduction) in the User Defined Help window.
  4. Select to edit the contents of the specific help topic. The Verify user login prompt is displayed.
  5. Enter a username in the Username field.
  6. Enter a password in the Password field.
  7. Select Login. The displayed topic RTF file is opened in WordPad.
  8. To save the changes, select Save.
  9. To save the file as a new RTF file, select Save As. The Save As window is displayed.

    All RTF files created and placed in \\rtehost\rte\proDASHelp\UserHelpFiles are automatically displayed in the User Defined Help drop-down menu. Saving RTF files to a different location will prevent the files from being displayed in the OHT.
