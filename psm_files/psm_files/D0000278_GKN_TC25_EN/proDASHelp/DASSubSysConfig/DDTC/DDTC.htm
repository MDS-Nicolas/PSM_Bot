Dynamic Data Telemetry Control (DDTC) Subsystem (Option*)

Overview

The DDTC Subsystem driver provides functions to open and close the communication over the telemetry control link to the telemetry devices. The DDTC Subsystem scans the status of the Telemetry Units, stores the status in CVT channels, and sends control requests to the Telemetry Units.

Operation

The proDAS DDTC Subsystem driver controls communications on the telemetry control link to the telemetry units.

proDAS transmits control commands to the STINCA units to:

  • select one of four banks of channels (labeled A, B, C and D)
  • automatically sets the gain for either or both groups of channels within the current bank based upon the selection change
  • set the gauge depolarisation ON or OFF
  • set the calibration signal ON or OFF
  • get the power status and air status

For parameter details, see Telemetry Control (STINCA) in DDS Subsystem.

proDAS transmits control commands to the DIGBERT receiver and wing box devices to:

  • get/set the tuning level for the receiver
  • get/select one bank of channels among 3 (labeled A, B, and C)
  • get the rotor power and temperature status
  • get/set the gauge depolarization ON or OFF
  • get/set the calibration signal ON or OFF

proDAS transmits control commands to the PINCA receiver and wing box devices to:

  • get/select one bank of channels among 4 (labeled A, B, C, and D)
  • get/set the tuning level for receiver A and B
  • get/set the gauge depolarization ON or OFF
  • get/set the calibration signal ON or OFF
  • get/set the power switch for the DIGBERT device ON or OFF
  • get/ramp the power level
  • get the coil loading
  • get the rotor power feedback, temperature status, and air on status
  • get/control the wing box power supply (STANDBY or ON)
  • get the UHF status

proDAS transmits control commands to the CHINCA units to:

  • get/select the desired channel number in the 1 - 24 range (using the control channels; CH_INCR, CH_DECR and CH_SEL)
  • get/set the MODE (Acceleration, Velocity, or Pressure) for the currently selected channel
  • get/set the GAIN (RANGE) for the currently selected channel
  • connect or disconnect the input for the current channel
  • get/set the Calibration (ON or OFF) of the CHINCA unit
  • get the Air status, and Power status of the CHINCA unit

For parameter details, see Telemetry Control (CHINCA) in DDS Subsystem.

proDAS transmits control commands to the FBOSTINCA unit to:

  • get/set one bank of channels among 4 (labeled A, B, C and D)
  • get/set the desired channel number (in the range 1-24)
  • get/set the GAIN (RANGE) for the currently selected channel (in the range 0-7)
  • get/set the calibration frequency (in the range 0-15)
  • get/set the gauge depolarization (ON/OFF)
  • get/set the Calibration (ON/OFF)
  • get/set the unit gauge trip (ON/OFF)
  • get the channel trip status (TRIP/NOT TRIP)

For parameter details, see Telemetry Control (FBOSTINCA) in DDS Subsystem.

proDAS transmits control commands to the FBOCHINCA unit to:

  • get/set the desired channel number (in the range 1-24)
  • get/set the GAIN (RANGE) for the currently selected channel (in the range 0-3)
  • get/set the MODE (Acceleration, Velocity, or Pressure) for the currently selected channel
  • get/set the calibration frequency (in the range 0-15)
  • get/set the Calibration (ON/OFF)
  • get/set the Calibration mode (ON/OFF)
  • get/set the gauge depolarization (ON/OFF)
  • get/set the unit gauge trip (ON/OFF)
  • get the channel trip status (TRIP/NOT TRIP)

For parameter details, see Telemetry Control (FBOCHINCA) in DDS Subsystem.

proDAS transmits control commands to the TINCA unit to:

  • get/set the desired channel number (in the range 1-24)
  • get/set the GAIN (RANGE) for the currently selected channel (in the range 0-2)
  • get/set the auto balance bridge command (ON/OFF)
  • get/set the reset auto balance bridge command (ON/OFF)
  • get the bridge circuit status (1/4, ½, full bridge)
  • get the resistance status (350Ω or 120Ω)
  • get/set the calibration frequency (in the range 0-15)
  • get/set the bridge polarization setting (0V, 2V, 5V, 10V)
  • get/set the Calibration mode (ON/OFF)
  • get/set the Calibration (ON/OFF)
  • get/set the gauge depolarization (ON/OFF)
  • get/set the unit gauge trip (ON/OFF)
  • get the channel trip status (TRIP/NOT TRIP)

For parameter details, see Telemetry Control (TINCA) in DDS Subsystem.

proDAS transmits control commands to the CWINCA unit to:

  • get/set the desired channel number (in the range 1-24)
  • get/set the GAIN (RANGE) for the currently selected channel (in the range 0)
  • get/set the test mode (ON/OFF)
  • get/set the unit gauge trip (ON/OFF)
  • get the channel trip status (TRIP/NOT TRIP)

Communication errors are reported with the proDAS Unified Event Log (UEL).

proDAS obtains the following general status information from the FBOSTINCA, FBOCHINCA, TINCA, and CWINCA devices:

  • get the unit id of the selected device
  • get the unit temperature
  • get the air supply indication (ON/OFF)
  • get the positive power indication (HIGH/LOW)
  • get the negative power indication (HIGH/LOW)
  • get the over temperature indication (ON/OFF)
