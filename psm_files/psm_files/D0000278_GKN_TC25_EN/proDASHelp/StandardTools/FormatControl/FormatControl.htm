Format Controls

Overview

Format Controls allow the user to set up the display formats of different data types. The format controls consist of:

  • Float Fixed Point Format
  • Float Scientific Format
  • Boolean Format
  • Integer Number Format
  • Integer Enumeration Format
  • Date Format
  • Time Format

Float Fixed Point Format

The Float Fixed Point Format allows the user to specify the number digits to be printed before and after the decimal for floating decimal point values.

Float Scientific Format

The Float Scientific Format sets the scientific format for floating point values. The Float Scientific Format is limited to six significant digits.

Boolean Format

The Boolean Format allows the specification of two string values which represent TRUE (non-zero) and FALSE (zero) of a boolean value. In a dialogue, a Combo Box is created which allows the selection of the value if the channel is editable.

Integer Number Format

This format allows the user to specify the number of digits which will be used for the channel value. Leading zeros will be ignored. For example, having 5 in the Number field allows numbers up to 99999 to be displayed.

This format supports both hexadecimal and binary numbers.


Integer Enumeration Format

The Integer Enumeration Format allows the user to define a list of values which are valid for the channel. In a dialogue, a Combo Box is created which allows the assignment of a value if the entry is editable. In the above example, the numeric values above 0 but less than 1 are represented by the text zero. Numeric values equal to or greater than 1 and less than 2 are represented by the text one. Numeric values equal to or greater than 2 but less than 3 are represented by the text three. To add values to the list:

  1. Select the enumeration boolean.
  2. Move the cursor into the combo box and enter the text value that represents values greater than or equal to 0 and less than 1.
  3. Type Ctrl/Enter to move the cursor to the next line.
  4. Repeat steps 2 and 3 until the desired number of enteries is reached.
  5. Select OK.

Date Format

The Date Format allows the user to specify the appearance of Date values (yy or yyyy = year, mm = month and dd = day).

Time Format

The Time Format allows the user to specify the appearance of Time values (HH = hour, MM = month, SS = seconds and sss = milliseconds).
