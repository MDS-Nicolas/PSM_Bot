Off-line Review of Unified Event Logs

Overview

The proDAS Unified Event Log (UEL) tracks all the events that occur while proDAS is running. All messages from the Real-Time Engine are stored in a single event log file. Text messages are written to the event log file whenever an event occurs (e.g.: starting test monitoring, taking a fullset, or registering an alarm).

A new event log is created and started whenever proDAS is reconfigured.

Event logs are saved to the hard disk of the Host Computer:

/users/RTE/logs/<enginename>/<testname>/event_logs/elYYMMDDHHmmss

where:

  • enginename is the name of the test engine
  • testname is the name of the test record
  • el identifies the file as an event log
  • YYMMDD is the date (in a year (YY), month (MM), and day (DD) format) that the log was opened
  • HHmmss is the time (in an hours (HH) (i.e.: 00-23), minutes (mm), and seconds (ss) format) that the log was opened

Off-line Access to an Event Log File

The Unified Event Logs can be reviewed off-line from any location as long as the operator has access to the UELDisplayServer.exe file. To review the event logs from previous engine tests:

  1. Select Start\All Programs\Accessories\Run from the Taskbar.
    The Run window is displayed.
  2. If the location of the UELDisplayServer.exe file is known, enter the location (e.g.: D:\proDAS\bin\UELDisplayServer.exe) in the Open: field.
    If the location of the UELDisplayServer.exe file is not known, go to Step 4.
  3. Append the command line flag /offline to the UELDisplayServer.exe file and go to Step 9.
  4. If the location of the UELDisplayServer.exe file is not known, select Browse... from the Run window.
    The Browse window is displayed.
  5. Select the file path (e.g.: proDAS) from the Look in: box on the Browse window.
    The contents of the proDAS folder are displayed in the Browse window.
  6. Select the file path (e.g.: bin) from the Look in: box on the Browse window.
    The contents of the bin folder are displayed in the Browse window.
  7. Select a file (e.g.: UELDisplayServer.exe) from the contents of the bin folder displayed in the Browse window.
    The file name is displayed in the File name: field when the file (e.g.: UELDisplayServer.exe) is selected.

  8. Select Open and go to Step 3.
  9. If the location of the Event Log file (e.g.: D:\proDAS\data\UEL Files\el030812164454.uel) is known, enter the location in the Open: field after the /offline flag.
  10. Select OK.
    The Unified Event Log (UELDisplay) window is opened.
  11. If the location of the log file was not entered in Step 9, an Open dialogue window is displayed.
  12. From the Open dialogue window, browse for the event log file using the Look in: box on the Open window and selecting the log file from the Open window display area.

    Log files saved through the UELDisplay have a .uel extension. Event logs created by the RTE do not have an extension.

  13. To view the files created by the RTE, click on the Files of type: box and select All Files (*.*).
    The display area of the Open dialogue box window is populated with all the log files in the selected folder.
  14. From the list of files in the Open window, select the file (e.g.: el030812164454.uel) to be opened.
  15. Select Open.
    The Unified Event Log Display (i.e.: UELDisplay) screen is opened.
