Channel Name Search (CNS)

Overview

The Channel Name Search (CNS) is a stand-alone application for searching references of channels in the XML configuration, scripts referenced by macros, RTD pages and Tabular Channel Display (TCD) files. The Channel Name Search (CNS) can be launched:

  • as a stand-alone application
  • from the Test Page by selecting from the Set Channel tab
  • from the Channel Editor by selecting Find Channel in proDAS from the Edit menu or from the context sensitive menu
  • from the Channel Editor Edit Channel window by selecting

The path to the Channel Name Search is defined in the ChannelEditor.ini file. If the path is not defined in the ini file, the Channel Editor startup directory is used.

Channel Name Search Window

The main window consists of:

  • Menus
  • Search Controls
    • a combined list and text box for specifying a channel name
    • a Find button
    • checkboxes and controls for selecting/deselecting data categories
  • Result Table
  • Status Bar

Menus

The main menu is divided into the following pull-down menus:

  • File
  • Search
  • Help

The File menu (displayed below) is described in the table below.

Menu or Column Description

The Search menu (displayed below) is described in the table below.

Menu or Column Description

Help

The Help menu (displayed below) is described in the table below.

Menu or Column Description
Getting Started... Opens the first page of the proDAS Integrated Help.
Topics Contains all specific help topics referring to the Channel Name Search.

Search Controls

A combined text and list box is available where the operator can specify a channel name. Initially, the list box contains the recently used channel names. After selecting a configuration, all available channels of the selected configuration are available.


Menu or Column Description
Channel Name Displays a list of available channel names.
Find Starts the search.
Numbers of occurrences per category Displays the number of hits for each category based on the last search displayed. Checkboxes are available to select/deselect categories for the specified channel name being searched.
All Selects all categories.
None Deselects all categories.
Invert Deselects all selected categories and selects all deselected categories.

Result Table

The result table displays the locations where the channel name was found.


Column Description
Category Displays the data category.
Item Name Displays the name of the entity within the data category.

For TCD files and RTD pages a file name is displayed.
Level Displays the level within the XML configuration.

Not applicable for TCD files and RTD pages.
Enabled Displays whether the item is enabled in the configuration.

Not applicable for TCD files and RTD pages.
Active Displays whether the item is active in the configuration.

Not applicable for TCD files and RTD pages.
Type Displays the item type.

Contents depends on the category. Does not apply to all categories.
File Path Displays the path to the macros, TCD files and RTD pages.
Location Describes the location of the channel reference within the item.
#Occurrences Describes the number of occurrences within macros, TCD files and RTD pages.

Status Bar

The status bar displays the currently selected configuration.


Performing a Channel Name Search

To perform a Channel Name Search:

  1. Launch the Channel Name Search from the MGT GUI Test Page (), Channel Editor (Find Channel in proDAS from the Edit menu or from the context sensitive menu) or Edit Channel window (). The Channel Name Search is displayed.
  2. From the Search menu, select Select Configuration.... The Select proDAS configuration window is displayed.
  3. Select the required configuration, select OK. The Channel Name Search window is displayed.
  4. Specify a channel name and search criteria (select/deselct desired categories), select Find. The search results are displayed in the Results table.
  5. From the File menu, select Save Results... to export the results.
  6. Import the CSV file into Excel (delimiter is tab).
