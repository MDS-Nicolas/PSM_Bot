Dynamic Data System (DDS) - Muller-BBM PAK MKII

Overview

The Dynamic Data System (DDS) is used to collect engine data that requires high sampling rates, such as rotational speed, vibration, and large pressure changes. The system acquires, monitors, and archives selected engine parameters during a test.

For example, a DDS may be used to acquire data on the following:

  • rotational speed
  • aircraft vibration monitoring
  • dynamic displacement
  • dynamic pressure, acceleration and velocity

See Operation for details on this test facility's DDS.
