Dynamic Data System (DDS) - Muller-BBM PAK MKII

Components

The Dynamic Data System (DDS) - Muller-BBM PAK MKII contains the following major components:

Component Description
PAK MKII Mainframe The PAK MKII Mainframe is a MF04 four slot mainframe with a VMEbus backplane. VMEbus is a mechanical and electrical standard. The PAK MKII uses this standard for its main communication bus. The mainframe contains the following VMEbus boards:
  • (1) VMEbus PQ30 G2 System Controller with 200 W Power Supply
  • (2) VMEbus SC42 G2 Signal Conditioning Boards
  • (3) VMEbus CHG42S G2 Charge Amplifier Signal Conditioning plugin
  • (2) VMEbus WSB42X G2 Wheatstone Bridge Signal Conditioning plugin
  • (1) VMEbus ICT42 G2 Voltage and Tacho Signal Conditioning plugin
  • (1) VMEbus IRG42 G2 Timing plugin

The mainframe is installed in client Rack Assembly AS3.
PAK Embedded Analyzer The PAK Embedded Analyzer is the independent software operating within the PAK MKII Mainframe which allows it to operate in a manner independent from the PAK 5.8 Software.
PAK 5.8 Software The PAK 5.8 Software is a PC-based application which allows for reording and processing of dynamic data. For details, see the PAK MKII Testbench System User Manual.

The PAK Data Server serves as the raw data reservoir that transfers accumulated data via the user network. It also serves as the engine for the PAK post-test analysis tools and the configuration system to load setups to the PAK MKII Mainframe.
ModBus2PAK Software The ModBus2PAK software is the main communication interface between proDAS and the PAK software.
PAK Workstation The PAK Workstation is the user interface to the PAK 5.8 Software. It consists of a key board, mouse and monitor installed on a console in the Control Room. KVM Extenders are used to make the link between the workstation on the console in the Control Room and the PAK workstation in equipment rack CM1 in the Computer Room.

For more information on the Dynamic Data System (DDS) - Muller-BBM PAK MKII components, see Vendor Documentation.
