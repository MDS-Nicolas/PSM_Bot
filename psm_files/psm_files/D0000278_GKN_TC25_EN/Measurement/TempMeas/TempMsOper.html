Temperature Measurement System

Operation

The Temperature Measurement System uses 192 channels that are scanned at up to 100 times per second (50 hertz bandwidth).

The Temperature Measurement System is ready for operation if no error messages appear in the Unified Event Log (UEL). Temperature scanning starts after proDAS is configured and scanning is started.

System measurements are displayed on the Real-Time Display Computer. The engineering unit (EU) conversion to either Celsius or Fahrenheit is performed by each scanner as defined in the Subsystem Editor. The Channel Editor assigns the temperature unit to each channel.

The network address for each scanner is manually configured using the scanner's web interface. For details, see Ex1048A User Manual.

Calibration

The channels for the EX1048 temperature scanners are defined in the VTI EX10xxA (VEXA) Subsystem. The exhaust thermocouple measurement channels are defined in the GASSVXI Subsystem. All channels are configured using the Channel Editor.

Individual sensors are configured with the Sensor Editor (the Sensor Editor also displays detailed sensor information).
