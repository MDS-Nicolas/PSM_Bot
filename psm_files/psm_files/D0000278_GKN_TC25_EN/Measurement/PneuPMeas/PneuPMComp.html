Pneumatic Pressure Measurement

Components

The Pneumatic Pressure Measurement System contains the following major components:

Component Description
NetScanner 9216 Pneumatic Intelligent Pressure Scanners (16 channels) The NetScanner 9216 pressure scanners are for measuring pneumatic pressure in the facility and the engine.

Depending on their firmware version, the older models of NetScanner Pressure Scanners may not be compatible with the proDAS software:

• 9016 pressure scanners: for scanners installed with a 1 Mbit flash chip a minimum firmware version of 2.42 is required, and for scanners installed with a 2 Mbit flash chip a minimum firmware version of 2.45 is required in order for proDAS to be able to communicate with the scanners. There is a command that can be used to confirm the flash size as follows: Send the ‘~ 2000’ command to the module in question. If the module returns 262144 as the first field in its response, then the flash size is 2Mbit.

• 9021 and 9022 pressure scanners: a minimum firmware version of 2.39 is required in order for proDAS to be able to communicate with the scanners.

• 9816 pressure scanners: a minimum firmware version of 2.32 is required in order for proDAS to be able to communicate with the scanners.

For more information on the Pneumatic Pressure Measurement System components, see Vendor Documentation.
