GKN Aerospace Test Cell 25

Overview

The Indoor Aero Engine Test Facility at the Trollhattan test site located in Trollhattan, Sweden is designed to:

  • rig engines in preparation for test
  • de-rig engines post test
  • allow production and limited development testing of engines

The test facility utilises MDS' proDAS data acquisition software and integrates with Dynamic Data Systems for precise, real-time engine test results.
