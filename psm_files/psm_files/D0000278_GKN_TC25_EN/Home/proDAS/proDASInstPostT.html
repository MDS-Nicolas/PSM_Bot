Transfer Diary

Transfer Diary prepares the Excel Test Diary Logsheet file and transfers it downstream to the RRDS System (Alice). The command line parameters consist of:

  • /mode - [ logsheet or "" | transfer | autotr ]
  • /test_cell_id - (integer) The ID of the current test cell
  • /test_id - (integer) The ID of the current test
  • /engine - (string) The name of the engine configured
  • /date - Event date or control keyword (auto, off, now)

The Date Filter command line parameter (/date) provides the ability to extract one day's worth of information. The date parameter can be:
- specific date: /date 10-31-2009
- yesterday: /date auto
- do not filter on date: /date off
- today: /date now
- if /date is not specified a pop-up window is displayed prompting the user to specify the parameters

View Diary
