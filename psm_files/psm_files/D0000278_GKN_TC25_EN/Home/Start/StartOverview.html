Getting Started

Overview

The MDS Product Support Manual (MDS PSM) is an on-line web based user friendly technical package offering concise and informative data supported by graphical illustrations and photographic images. Coupled with the mouse point and click function, the user is capable of navigating through the manual. Specific text items may be linked to other areas of this document (Hyperlinks), enabling the reader to jump directly to another source of information that best describes or illustrates specific information about the MDS delivered products and associated systems.

This versatile integrated web-based utility comes with dynamic tools such as the Adobe Acrobat Reader which enables the user to open, navigate and print pdf files, and JavaScript, which are software modules that add features to a system allowing access to specific file formats e.g. various graphics and 3D file formats.

The standard resolution size for the MDS Product Support Manual is 1280x1024, with the viewing screen area adapting to the various monitor resolution sizes. For smaller viewing areas, scroll bars are automatically placed along the bottom (horizontal left and right scrolling) and the right side of the screen (vertical upward and downward scrolling). The MDS Product Support Manual creates a unique, user friendly learning environment for all levels of users.

The following is the minimum hardware requirement to successfully run and operate the MDS Product Support Manual:

  • an Intel Pentium processor or equivalent, 600 MHz processor, 128 MB RAM, DVD-ROM drive
  • video card must be capable of running at 1280x1024 pixels, True Color (recommended for optimal viewing)
  • 800 x 600 VGA display or better

The following is the minimum software requirement for each computer and is required to run and operate the utility:

  • Google Chrome
  • Adobe Acrobat Reader
  • Zoom Search Engine

Hyperlinks

Hyperlinks, denoted by coloured underlined text ( Hyperlinks ), are text items that are linked to another page or paragraph that directly relates and supports the hyperlinked item. The Hyperlink allows for rapid transport from one location in the MDS Product Support Manual to another. By placing the mouse cursor over a hyperlink, the cursor image changes from an arrow to a white hand with a pointing finger. Clicking the cursor while over a hyperlink opens the page containing the information indicated by the hyperlink in the main viewing area. All topics listed in each information module are linked to the related subject matter. Click on any hyperlinked topic to display the subject in the main viewing area.

Hyperlinks found within the system write-ups display related information. The link may be specific (i.e.: Click here to...) or implied and embedded in the main text. In either case, click on the link to display the associated information in the main window.

It is recommended that your browser history be cleared regularly to reset all links back to the active colour. See the How To section in Operation for details.
