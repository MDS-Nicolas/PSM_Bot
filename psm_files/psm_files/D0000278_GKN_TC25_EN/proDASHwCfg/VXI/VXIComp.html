VXI Hardware Interface

Components

The VXI System consists of a VXI chassis which contains various plug-in cards.

The VXI System is located in the Instrumentation Room in rack CM2, slot A5.

The VXI Subsystem in proDAS communicates to the VXI chassis through a PCI-MXI-2 card in the PCI Expansion Chassis of the Host Computer. An MXI-2 cable connects the PCI-MXI-2 card to a VXI-MXI-2 Extender Card installed in the VXI chassis.

The VXI Hardware Interface System contains the following major components:

Component Description
VXI Chassis The CT-400 C-Size VXI mainframe chassis provides a communications backplane for the VXI cards and a communications interface to proDAS. The chassis is mounted in an equipment rack to provide easy access for servicing and maintenance. The chassis contains a power supply, which provides power to the chassis and to all the VXI cards installed in the chassis. Fans within the chassis cool the power supply and the cards.
VT1413C 64-Channel High-Speed Scanning A/D Converter Card The VT1413C 64-channel high-speed scanning A/D converter card converts analogue signals to digital signals. It provides:
  • high-speed scanning
  • 16-bit resolution
  • 0.02% full scale accuracy
  • 4 mV to 16 V full-scale input
  • 64 kSa dual-ported FIFO buffer
  • current value buffer for on-line monitoring
  • automatic self-calibration of the complete path for every channel
  • auto zero
  • built-in self-test diagnostics
VT1503A Eight-Channel Programmable Filter and Gain SCP The VT1503A eight-channel programmable filter and gain SCP conditions up to eight channels.

Each channel has a programmable low-pass passive filter with cutoff frequency settings of 2, 10, and 100 Hz. The card converts channels at up to 200 Hz scan rate. The programmable input amplifiers for each channel have gain settings of 1, 8, and 64. The card provides input over-voltage protection and open transducer detection for each channel.
E1458A 96-Channel Digital Input/Output (I/O) Module The E1458A 96-channel digital input/output (I/O) module has twelve ports intended for digital input/output control in electronic environments. It is compatible with both TTL levels (0-5 volts) and CMOS levels (using an external pull-up). It contains twelve 8-bit input or output ports for data acquisition, plus six handshake lines for each port. Each port can be configured for input or output using the Channel Editor but, all channels on a port must have identical configuration, input or output.

Ribbon cable connects the input/output data lines to a terminal panel containing optical isolators. All inputs and outputs used in this facility for the E1458A card are optically isolated. The output data lines can be configured as decimal, binary, octal, or hexadecimal. The read or write operation of a data port is carried out on a per-port basis. This implementation controls and monitors each bit.
PCI-MXI-2 extender card The VXI Subsystem in proDAS communicates with the VXI chassis through a PCI-MXI-2 card in the PCI expansion chassis of the Host Computer. A MXI-2 cable connects the PCI-MXI-2 card to a VXI-MXI-2 extender card in the VXI chassis.
777185-01 National Instruments PCI-MXI-2 Card Kit
777164-02 The MXI-2 cable connects the VXI chassis to the PCI expansion chassis.
CA801-2M The L-Com 2m SCSI-2 cable connects the VTI1413C card in the VXI chassis to the VXI terminal panel. For connection details, see A-0000237 Assembly-Wiring, GKN, Test Cell 25, Analog Inputs.
CA801-3M The L-Com 3m SCSI-2 cable connects the VTI1413C and the VTI1415A cards in the VXI chassis to the VXI terminal panel. For connection details, see A-0000237 Assembly-Wiring, GKN, Test Cell 25, Analog Inputs.
A-0000278 The digital I/O ribbon cable assembly connects the E1458A card in the VXI chassis to the digital input/output chassis.
