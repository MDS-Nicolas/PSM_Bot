Analogue Inputs

Overview

Analogue Inputs measure various analogue voltage and current signals originating from the engine and facility systems. Transducers on the engine or facility system transmit the analogue signals directly from the source to VXI interface terminals installed in an Equipment Rack. The VXI Terminal Chassis contains custom MDS VXI Interface circuit boards. The signal is then transmitted to the VXI Main Chassis. The VXI Main Chassis contains multi-channel low pass filter Signal Conditioning Plug-in modules (SCPs) and programmable filter and gain SCPs which are mounted on multi-function measurement cards. The VXI Main Chassis is installed in an Equipment Rack, which is located in an Instrumentation Room in the facility. Analogue Input signals are processed by proDAS as GASSAI or GASSVXI channels. Refer to Sensor Calibration Application (SCA) for information on auditing and calibrating VXI channels.

For each input channel that is measuring current, a 250 ohm or 500 ohm resistor must be installed on the channel input of the VXI Interface card (in the VXI Terminal) before the signal is transmitted to the VXI Main Chassis.

The same hardware is used for both voltage and current signals. Adding a 500 ohm resistor to the signal path enables the 4-20 mA current to be measured as 2-10 VDC. A 250 ohm resistor enables the 4-20 mA current to be measured as 1-5 VDC.

Analogue Input data is displayed on a Real-Time Display Computer. The channels are defined within the GASSVXI Subsystem. The channels are configured with the Channel Editor, and individual sensors are configured with the Sensor Editor.

The Sensor Editor also displays detailed sensor information.
