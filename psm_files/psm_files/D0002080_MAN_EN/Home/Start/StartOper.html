Using the MDS Product Support Manual

The MDS Product Support Manual (MDS PSM) is the user's primary source of reference information for the MDS delivered products. A user friendly NavMenu is always available at the left side of the viewer's screen. The user is able to quickly navigate via Hyperlinks (to jump) from one information module to another or, where applicable, to easily travel within the currently opened information module. The NavMenu is equipped with clearly marked main categories for quick navigation through the MDS Product Support Manual. A category represents a group of technical or non-technical information modules.

When a category has been selected, the respective group of topics is displayed under the category. When a topic has been selected (user has clicked on the topic text), the subject matter list is displayed. When selected, the subject matter is displayed in the main viewing area.

NavMenu

The NavMenu is comprised of categories. These categories may be technical or non-technical. Each category contains topics. Each topic contains subject matter related specifically to that topic. The NavMenu expands and contracts (like an accordion) to display more or less information on the selected category or topic. The NavMenu has a dark grey background colour and white text.

NavMenu Main Categories

Selecting a main category displays the topics available for that main category, as shown below. The selected main category background colour changes from dark grey to blue, and the plus sign icon changes to a negative sign, indicating that the category has been expanded. Available topics for the selected category are displayed with a white background colour and black text.

NavMenu with Topic Selected

Selecting a topic displays the subject matter available for that topic. The selected topic background colour changes from white to green, the text colour changes from white to black, and the plus sign icon changes to a negative sign, indicating that the topic has been expanded. Available subject matter is displayed with a light grey background colour and black text.

NavMenu with Topic Subject Matter Displayed

Selecting the subject matter displays the related content in the main viewing area. The selected subject matter background colour changes from grey to blue, and the text colour changes from black to white.

NavMenu with Subject Matter Selected and Content Displayed in Main Viewing Area

NavMenu Main Categories

The NavMenu is divided into different categories, based on the client facility. Each category contains a group of related information modules that contain topic indexes to navigate directly to the detailed subject matter. The NavMenu displays the main categories supplied for the facility. These systems may include:

  • Search
  • Home
  • proDAS Integrated Help (if provided)
  • References

Search

The MDS Product Support Manual contains a user friendly search feature. When the feature is selected, a customised search page is displayed. Search criteria (words or phrases) are entered into the page, and the feature displays information that is found within MDS Product Support Manual which contains the criteria.

Home

The Home category contains the Getting Started module and the Your proDAS System module. The Getting Started module is a user friendly introduction to the definitions and operation of the MDS Product Support Manual and the Your proDAS System module (if provided) is a breakdown of a facility's proDAS system configuration.

proDAS Integrated Help (if provided)

This category launches the proDAS Integrated Help. Packaged with every proDAS installation, the proDAS Integrated Help is the user's primary source of information for the installed proDAS system. The Help focuses on generic proDAS features such as Testing with proDAS, Subsystem Configurations, Standard Tools and Utilities, Configuration Tools and Custom Applications.

References

A series of on-line reference documents is included in the MDS Product Support Manual to further guide and inform the user. These reference documents may include, but are not limited to, the following:

  • Calibration information
  • Maintenance (Master List)
  • MCLs (Master Channel Lists)
  • MDS Engineering Drawings (Master List)
  • Recommended Spare Parts Lists (RSPLs) (Master List)
  • Vendor Documents (Master List)
